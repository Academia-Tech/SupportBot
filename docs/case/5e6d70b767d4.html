Проблема

El usuario tiene dificultades para conectar un router Fritzbox de fibra con su conexión Movistar, preguntando sobre el tipo de cable (SC-SC vs LC/SC) y la compatibilidad con IPTV. A pesar de usar el cable recomendado, la conexión no funciona.

Рішення

Para conectar un router Fritzbox Fibra en España (ej. con Movistar), se necesita un cable (latiguillo) de fibra con conectores LC/APC y SC/APC. Es crucial usar el módulo SFP GPON original de Fritzbox, ya que uno genérico o de otra marca podría no funcionar. El usuario descubrió que su equipo de segunda mano venía con un módulo AON en lugar del GPON necesario. Tras seguir el consejo, el usuario ha encargado el módulo GPON correcto y está a la espera de recibirlo para probar la solución.
