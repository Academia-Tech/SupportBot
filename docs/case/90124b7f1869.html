Para tantos usuarios y el tipo de trabajo, lo primero que te sugeriría es tener una segunda fibra, a ser posible de otro operador, por si algún proveedor tiene alguna caída. Además de ser una fibra backup de la otra fibra, lo ideal es que trabajen conjuntamente de forma balanceada y así poder atender momentos de pico de tráfico y superar el Giga que da una única línea.

Lo ideal sería tener un router profesional y puntos de acceso wifi, pero para esto necesitas al menos una persona (mejor dos) en la empresa que entienda bastante o bien, que subcontrates la instalación y el mantenimiento.

No sé si te encaja, pero este modelo de la gama profesional de Asus destinado a la pequeña o mediana empresa, no requiere de ser un especialista a la hora de configurarlo, se configura de forma similar que cualquier router de la marca y es compatible con doble wan, admite tener varios routers para tener wifi mesh, posibilidad de varias wifis separadas con Vlan, es capaz de gestionar 2.5Gb, tiene VPN para acceso remoto, buena wifi categoría AX7800, etc., aquí puedes ver las características https://www.asus.com/es/networking-iot-servers/business-network-solutions/asus-expertwifi/asus-expertwifi-ebm68/
