How to Reconnect Tasmota Device to Wi-Fi After Changing SSID

Case 27219c23be8c · tasmota

#Wi-Fi#SSID#reset#AP mode#device recovery#Ssid2#configuration

Проблема

A user changed their router's SSID and needed to know how to get their Sonoff Dual R4 device back into AP mode to reconfigure the Wi-Fi credentials. The standard device reset procedure of power cycling 6+ times did not work for them.

Рішення

Two solutions were provided to recover a Tasmota device after an SSID change: 1. **Device Recovery:** A user suggested a hard reset by power cycling the device 6 or more times to enter recovery mode. When that failed, they provided a link to the official Tasmota Device Recovery documentation for more methods: `https://tasmota.github.io/docs/Device-Recovery/`. 2. **Non-destructive Method:** Another user suggested a safer method: temporarily change the router's SSID back to the old one. Let the Tasmota device connect, then use its Web UI to configure the new SSID and password in the `Ssid2` field. After saving, the router's SSID can be changed back to the new one, and the device will connect.
