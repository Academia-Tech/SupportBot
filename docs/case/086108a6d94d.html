#Omada#Switches#Red Empresarial#Infraestructura de Red#Fibra#UTP#TP-Link

Проблема

El usuario necesita montar una infraestructura de red Omada para una empresa con 100-150 clientes. Busca recomendaciones para switches Omada que actúen como core (con fibra) y switches de distribución (con fibra y UTP) para conectar puntos de acceso y cámaras.

Рішення

Para la selección de switches Omada en una red empresarial, se recomienda revisar los productos disponibles en las webs oficiales de Omada y Ubiquiti, y considerar la marca Ruijie Reyee (con videos de @macjosan). Es crucial que todos los equipos sean de la misma marca (Omada o Ubiquiti). Para necesidades específicas o proyectos grandes, donde se requiere una topología de red detallada, se aconseja consultar a un profesional. Se sugiere que el mismo @macjosan podría atender este tipo de consultas.
