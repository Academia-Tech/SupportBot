Налаштування стабілізації по двох осях (pitch і roll) для гімбалу caddxfpv gm2 на ArduCopter

Case d1a11e8f1f6d · ua_ardupilot

#gimbal#caddxfpv gm2#стабілізація#pitch#roll#ArduCopter#PWM

Проблема

Користувач не міг налаштувати стабілізацію по осі roll для гімбалу caddxfpv gm2 на ArduCopter (версії 4.6.2 та 4.5.7). Стабілізація працювала лише по осі pitch, хоча на прошивці iNav все працювало коректно по обох осях. Проблема полягала у неправильному підході до налаштування через параметри Mount.

Рішення

Рішення полягало у відмові від налаштування через Mount (`mnt1_type`). Замість цього, стабілізація керується безпосередньо PWM-сигналами. Користувач user_2c254dce42 пояснив, що для ввімкнення стабілізації по обох осях (pitch і roll) необхідно подати на вхід PWM1 гімбалу сигнал зі значенням 2000 мкс. Вхід PWM3 використовується для ручного керування нахилом по pitch. Були надані наступні значення для PWM1: - 2000 мкс: стабілізація по pitch і roll. - 1500 мкс: стабілізація тільки по pitch. - 1000 мкс: режим прибирання вібрацій без стабілізації. Автор питання підтвердив, що це рішення спрацювало.
