Configurar Mini PC para que se encienda automáticamente tras un corte de luz

Case aa2fc8daee40 · domotica_es

#mini pc#bios#corte de luz#arranque automático#home assistant#hardware#power loss#sai

Проблема

Un usuario que ha migrado su instalación de Home Assistant a un Mini PC se encuentra con que el equipo no se reinicia solo después de un corte de luz, requiriendo un encendido manual. Busca una solución para que el arranque sea automático.

Рішення

La solución consiste en modificar una opción en la BIOS/UEFI del Mini PC. Se debe buscar en los menús de gestión de energía (Power Management) una opción similar a 'Restore on AC Power Loss', 'AC Power Recovery' o 'After Power Failure' y configurarla en 'Power On' o 'Last State'. Varios usuarios confirman este método y aportan una lista de nombres comunes para esta opción en diferentes marcas (Dell, HP, Lenovo). Como alternativa, si la BIOS no dispone de esta función, se puede utilizar un SAI (Sistema de Alimentación Ininterrumpida).
