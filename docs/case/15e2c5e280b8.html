Problemas de conexión al sustituir router de Digi por un TP-Link

Case 15e2c5e280b8 · naseros

#digi#tp-link#router neutro#pppoe#vlan#clonar mac#fibra óptica

Проблема

Un usuario intenta sustituir el router de Digi por un router neutro TP-Link. A pesar de haber configurado los datos PPPoE y la VLAN proporcionados por Digi, no consigue tener conexión a internet.

Рекомендація (не підтверджено)

Se sugieren varias posibles soluciones y pasos de diagnóstico para conectar un router TP-Link en una línea de Digi: 1. **Clonación de MAC**: Se menciona la posibilidad de que Digi requiera clonar la dirección MAC o el número de serie del equipo original para autenticarse. 2. **Introducción de credenciales**: Se sugiere que el problema podría estar en la introducción de las credenciales PPPoE. 3. **Configuración DHCP**: Se recomienda activar y configurar el DHCP en el router, estableciendo DNS primarios (Cloudflare) y secundarios (Google), y esperar un minuto tras cada cambio. Tras no lograr una solución con estos pasos, se recomienda como último recurso seguir un video tutorial para empezar la configuración desde cero: https://www.youtube.com/watch?v=dobhBu8DemU&t=993s&ab_channel=NASeros. El usuario agradece la ayuda.
