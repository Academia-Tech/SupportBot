Flashing and Configuring LC Technology 2-Channel 5V WiFi Relay

Case 5dc9e76af43c · tasmota

#LC Technology#relay#ESP-01#Tasmota#flashing#template#Blakadder#serial relay

Проблема

A user was trying to install Tasmota on an LC Technology 2-channel 5V WiFi relay module (ESP-01 based) and was having trouble getting it to work using the standard documentation.

Рішення

For troubleshooting an LC Technology serial relay board (like the 2-channel or 4-channel versions), several steps were discussed. Initial suggestions included changing the serial speed (e.g., `baudrate 9600`) or using a specific firmware fork for serial relays (`NicolasBernaerts/tasmota/tree/master/`). Further diagnostics from the user showed that the LED behavior (solid red/blue) was the same with or without the ESP module installed, and the `SerialSend5 A00101A2` command failed to activate the relay. Based on this, an additional recommendation was made to download the datasheets for the board's components, identify the power pins, and use a multimeter to verify that all power supply lines are stable and providing the correct voltage.
