Проблема

Un usuario informa que después de realizar varias actualizaciones en Home Assistant, su red Zigbee2MQTT dejó de funcionar. Intentó solucionar el problema restaurando una versión anterior de Home Assistant, pero esto no resolvió el fallo. Finalmente, tuvo que volver a emparejar todos sus dispositivos para que el sistema volviera a funcionar.

Рішення

La solución final al problema fue re-emparejar todos los dispositivos Zigbee. Sin embargo, se compartió una medida preventiva crucial para el futuro: se recomienda encarecidamente hacer una copia de seguridad manual de los archivos de configuración de Zigbee2MQTT, que se encuentran en la carpeta `share/zigbee2mqtt/`. Esta acción es fundamental porque las copias de seguridad estándar de Home Assistant no siempre incluyen toda la configuración de Z2M, y tener esta copia manual puede evitar tener que re-emparejar todos los dispositivos después de una actualización fallida o un problema en el sistema. El usuario agradeció el consejo.
