Проблема

El usuario busca soluciones para controlar dos persianas con un solo relé y opciones de interruptores, incluyendo la posibilidad de no usar interruptores físicos, y cómo manejar la instalación en un espacio reducido.

Рішення

Es posible usar un Sonoff 4CH (flasheado con Tasmota) para controlar dos persianas, configurando un relé para la activación y otro para el sentido (modo Switchmode 1 o 2). El usuario original confirmó que esta opción le parece útil. **Advertencias y alternativas**: * Varios usuarios desaconsejan fuertemente el uso del Sonoff 4CH para persianas debido a la falta de enclavamiento entre contactos (riesgo de cortocircuito al activar subida y bajada simultáneamente) y la ausencia de control porcentual preciso (solo se basa en finales de carrera o tiempo). * Se recomienda encarecidamente usar dispositivos específicos para persianas, uno por cada persiana, para mayor seguridad y funcionalidad. Los atajos no son camino de rosas. * Si se usa el Sonoff 4CH, se sugiere flashearlo con Tasmota para un mejor control y evitar la dependencia de eWeLink. * Para la instalación, se aconseja usar una caja de empalmes adecuada en lugar de dos cajas de mecanismos. * Con ESPHome o Tasmota, se podría lograr una posición porcentual basada en el tiempo, aunque no es lo ideal.
