Налаштування скрипту для відправки сповіщень з MikroTik у Telegram

Case e020dae74f7b · mikrotik_ua

#mikrotik#routeros#script#telegram#api#notification#alert#tool fetch

Проблема

Користувач намагається налаштувати відправку сповіщень з роутера MikroTik у Telegram, зокрема про виявлення неавторизованих DHCP-серверів. Виникають труднощі з правильним синтаксисом команди `/tool fetch`, форматом токена бота, ID чату та екрануванням символів у RouterOS.

Рішення

Для відправки сповіщень в Telegram з MikroTik було надано декілька варіантів скриптів та виявлено ключові помилки. Основні рекомендації: 1. Токен бота повинен починатися з префіксу `bot` (наприклад, `bot12345:ABC-DEF`). 2. Для відправки у груповий чат `chat_id` має починатися зі знаку мінус (`-`). 3. Рекомендовано використовувати змінні для токена та ID чату. 4. Наведено декілька робочих прикладів команди `/tool fetch`, включаючи сучасний синтаксис з `http-method=post` для RouterOS v7, що краще обробляє спеціальні символи.
