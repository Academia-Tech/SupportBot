#uncontrolled ascent#flyaway#vibration#notch filter#INS_HNTCH_REF#PID tuning#Stabilize

Проблема

Після скидання двох вантажів дрон раптово і дуже швидко набирає висоту, після чого робить різкий маневр (переворот) і падає. Політ відбувався в режимі PosHold.

Рекомендація (не підтверджено)

Проблема, ймовірно, пов'язана з вібраціями та некоректними налаштуваннями фільтрів. Рекомендовано: 1. **Увімкнути динамічний Notch-фільтр:** Перевірити параметр `INS_HNTCH_REF` і встановити його в `1` для активації фільтра на основі телеметрії ESC. 2. **Перевірити пропелери:** Раптовий пік вібрацій (наприклад, на 64 Гц, як видно з логу) може бути викликаний пошкодженим пропелером. 3. **Використовувати режим Stabilize:** Для перших польотів та перевірки базової стабільності краще використовувати режим Stabilize, оскільки режими з утриманням позиції (AltHold, Loiter, PosHold) вимагають додаткового налаштування. 4. **Зменшити PID:** Для першого польоту можна профілактично зменшити значення PID у 3-4 рази, щоб уникнути сильних осциляцій.
