Controlling Tasmota Shutters with Alexa Voice Commands

Case 40617066d235 · tasmota

#alexa#shutter#voice control#tasmota rules#pwm#dimmer#virtual device

Проблема

A user asked for a way to control their Tasmota-based shutters using Amazon Alexa for voice commands like 'open' and 'close'.

Рішення

The user implemented a successful solution by creating a virtual dimmer in Tasmota and linking it to the shutter's position. The steps are: 1. Create a virtual PWM dimmer by assigning the `PWM` component to an unused GPIO pin in the Tasmota template. 2. Use the Tasmota rule `on dimmer#state do ShutterPosition %value% endon` to synchronize the shutter's position with the dimmer's percentage value. 3. In the Alexa app, create routines to alias voice commands like "open shutter" to "set the dimmer to 100%" and "close shutter" to "set the dimmer to 0%". This method allows for full open/close control as well as setting partial positions (e.g., "Alexa, set shutter to 50%").
