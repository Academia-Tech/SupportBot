ESP8266 Displays Apache2 Default Page

Case c5291b6b6b04 · tasmota

#ESP8266#Network#IP Conflict#Troubleshooting#Web Interface#Apache2

Проблема

An ESP8266 device unexpectedly displays the default Apache2 web page on its main menu.

Рекомендація (не підтверджено)

The issue of an ESP8266 device's web UI showing an Apache2 default page was initially suspected to be an IP address clash on the network. However, after further troubleshooting which included checking ARP tables and powering down other network devices, the user discovered the root cause was a corrupted browser cache on their phone. Clearing the phone's browser cache resolved the problem completely.
