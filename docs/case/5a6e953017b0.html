Recomendaciones para ampliar la señal de una red Zigbee a una zona lejana

Case 5a6e953017b0 · domotica_es

#Zigbee#Repetidor#Router#SLZB-06#Sonoff#Cobertura#Red Mesh#Ethernet

Проблема

Un usuario con una red Zigbee de 52 dispositivos, coordinada por un Sonoff, necesita extender la cobertura de la señal a una zona distante como el jardín. Busca consejos sobre amplificadores o repetidores efectivos para lograr una conexión estable.

Рішення

Se recomiendan varias opciones, destacando el uso de un dispositivo SMLIGHT SLZB-06 configurado en modo router. Este dispositivo es ideal porque se conecta a la red por cable Ethernet (incluso con PoE), lo que permite ubicarlo en un punto intermedio óptimo para extender la malla Zigbee. Se confirma que es compatible con un coordinador Sonoff, pudiendo flashear el firmware de router en el SLZB-06 para que actúe como extensor. El usuario que preguntó confirmó que la solución le pareció adecuada y procedería a implementarla.
