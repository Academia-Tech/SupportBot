Warning and Alternatives for WVC Solar Inverters

Case 5937fb79f55e · tasmota

#solar inverter#WVC#Hoymiles#product recommendation#hardware warning#MPPT#ESP8266

Проблема

A user inquired about the feasibility of flashing Tasmota onto a WVC series WiFi-enabled solar inverter (e.g., WVC-600) to gain local control and avoid using the manufacturer's cloud service.

Рекомендація (не підтверджено)

A strong warning was issued against WVC inverters, describing them as a 'scam' and highly inefficient. They are significantly underpowered (e.g., a 1600W model is required to achieve 800W output) and their MPPT (Maximum Power Point Tracking) implementation is very poor, leading to over 30% power loss. Alternatives like micro-inverters from reputable brands (e.g., Enphase, Hoymiles) or central inverters with MPPT were recommended. It was also noted that the LPT270-0 Wi-Fi/Bluetooth module used in these inverters is based on a GENIE+ chip and cannot be simply reflashed. Further technical details indicate that the LPT270-0 module uses a GENIE+ chip and communicates via a simple set of AT commands. Messages are prefixed with '+', replies are '+ok' or '+ok=send failed', and connection status updates are '+ILOPCONNECT=WIFI_CONNECT'. Inverter reports include PV, AC, and temperature data via AT+SENDICA commands.

Short update to WIFI enabled WVC inverters: They use an LPT270-0 WIFI/Bluetooth module that is based on a GENIE+ chip (never heard of that manufacturer) - so sadly this cannot simply be reflashed...

But they use a relatively simple set of AT commands to the module, while the module prefixes every message with a '+' character, replies every command back and sends "+ok" or "+ok=send failed" and sends its WIFI and server connection status updates as "+ILOPCONNECT=WIFI_CONNECT" / "+ILOPCONNECT=WIFI_CONNECT"

All the inverter reports are the following two messages:
AT+SENDICA=property,PV_Volt,19.0,PV_Current,0.00,PV_Power,0.0,AC_Volt,0.0,AC_Current,0.00,Out_Power,0.0,Temperature,34.0,Power_adjustment,100,Energy,0.00
AT+SENDICA=property,PowerSwitch,0,Plant,0.00,Emission,0.00,Time,30,P_adj,0,TEMP_SET,66
