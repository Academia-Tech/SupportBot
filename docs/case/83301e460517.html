Cómo conectar un interruptor WiFi en un circuito conmutado de pasillo

Case 83301e460517 · domotica_es

#Electricidad#Interruptor WiFi#Conmutada#Shelly#Cableado#Instalación#Neutro

Проблема

Un usuario necesita ayuda para instalar un interruptor WiFi en un pasillo con dos interruptores (circuito conmutado). Al abrir uno de los mecanismos, se encuentra con un cableado que no entiende y ya ha quemado un interruptor intentándolo.

Рекомендація (не підтверджено)

Varios usuarios colaboran para diagnosticar el problema de un interruptor WiFi en un circuito conmutado. Se concluye que el interruptor del usuario no es adecuado porque requiere un cable neutro, que no está disponible en la caja, y no está diseñado para conmutación. La solución recomendada de forma unánime es utilizar un relé de la marca Shelly, calificada como una 'apuesta segura'. Se especifica que si no se puede pasar un cable neutro, se debe usar un switch o relé 'sin neutro'. Se aclara que un solo Shelly es suficiente y debe instalarse en la última caja de registro antes del punto de luz. Se comparten vídeos de instalación como referencia, aunque se advierte que los modelos sin neutro pueden no funcionar tan bien como los que sí lo usan.
