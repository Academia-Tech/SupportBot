Проблема

На відео показано 10-дюймовий дрон, який при спробі зльоту демонструє сильну нестабільність, осциляції та неконтрольований дрейф. Дрон не може стабілізуватися в повітрі. Проблема, ймовірно, пов'язана з некоректними налаштуваннями фільтрів та PID для даної конфігурації.

Рекомендація (не підтверджено)

Проблема полягає у відсутності налаштувань фільтрації. Надано повний набір рекомендованих параметрів для початкового налаштування фільтрів для 10-дюймового дрона, включаючи `INS_ACCEL_FILTER`, `INS_GYRO_FILTER` та параметри Harmonic Notch Filter (`INS_HNTCH_*`) на основі телеметрії ESC (режим 3). Також надано параметри для активації Bi-directional DShot (`SERVO_BLH_AUTO`, `SERVO_BLH_BDMASK`). Ці параметри можна зберегти у файл `.param` та завантажити в Mission Planner.
