Проблема

Un usuario solicita recomendaciones de gestores de contraseñas.

Рішення

Varios usuarios recomiendan Bitwarden, incluyendo la opción de autoalojamiento (Vaultwarden) y la posibilidad de autoalojar la versión oficial. Se destaca que Bitwarden nunca ha tenido brechas de seguridad. También se menciona Keepass como una buena alternativa, confirmando que está disponible para Android (se recomienda 'Keepass2Android Password Safe'). Se explica que Bitwarden ofrece sincronización en la nube, mientras que Keepass requiere más configuración para la sincronización local. Se aconseja probar ambos para decidir cuál se adapta mejor a las necesidades del usuario. El usuario que preguntó decide usar Bitwarden tras consultar a un experto.
