Case fd86c2ba51d5 · domotica_es

Проблема

Después de un reinicio del router o un corte de internet, la interfaz web de Home Assistant deja de ser accesible localmente, aunque otros servicios como Samba o Wireguard siguen funcionando. Para recuperarla, es necesario un reinicio completo del sistema.

Рішення

El problema fue identificado y resuelto por el propio usuario. La causa era un conflicto generado por tener instaladas a través de HACS dos versiones o "forks" de la misma integración personalizada (en este caso, para termostatos Uponor). Al eliminar una de las integraciones duplicadas, el conflicto desapareció y Home Assistant volvió a ser estable tras los reinicios del router.
