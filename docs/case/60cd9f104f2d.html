Sending Sensor Data from Tasmota to Google Sheets

Case 60cd9f104f2d · tasmota

#Google Sheets#Tasmota#ESP32#webquery#data logging#API#scripting#rules

Проблема

A user asked for a method to send sensor data from an ESP32 device running Tasmota directly to a Google Sheets document for data logging purposes.

Рішення

It was confirmed that this is possible using Tasmota's `webquery` command. The solution involves two main parts: 1. On the Google side: Create a Google Apps Script web app that accepts data via GET requests and writes it to a sheet. Two tutorials were provided for this: https://iotdesignpro.com/articles/esp32-data-logging-to-google-sheets-with-google-scripts and https://www.electroniclinic.com/send-data-from-google-spreadsheet-or-google-sheets-to-esp8266-or-esp32/. 2. On the Tasmota side: Use a rule or a Berry script to construct and execute a `webquery` command that sends the data to the web app's URL. An example command was given: `webquery https://script.google.com/macros/s/YourToken/exec?1_Spalte=100&2_Spalte=200 GET`. The user acknowledged the solution and intended to try it.
