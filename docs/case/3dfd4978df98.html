Calibrating WCS 1800 Current Sensor with Tasmota Rules

Case 3dfd4978df98 · tasmota

#rules#analog sensor#calibration#scale command#WCS 1800#current sensor#Wemos D1

Проблема

A user was trying to read a WCS 1800 current sensor on a Wemos D1's analog pin (A0). They had the raw ADC values for 0A (845) and 12A (1124) but was struggling to create a Tasmota rule to correctly scale the reading and publish it as Amps. Their initial attempts with complex rules and math expressions were failing or causing the device to restart.

Рішення

After initial complex rule attempts failed, the user found a simplified solution that worked. By using the `scale` command directly within a one-shot rule, they were able to map the raw ADC range to the desired Ampere range. The final working rule was: `on ANALOG#A0 do scale1 %value%,845,1124,0,12 endon` This approach avoids complex variables and math expressions, providing a stable way to perform linear calibration directly on the device.
