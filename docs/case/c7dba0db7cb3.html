Проблема з отриманням телеметрії напруги та струму з ESC NeutronRC AM32

Case c7dba0db7cb3 · ua_ardupilot

#ESC#телеметрія#NeutronRC#AM32#напруга#датчик струму#BATT_MONITOR#UART

Проблема

Користувач зіткнувся з проблемою, що ESC NeutronRC E-AM32 70A не передає коректні дані про напругу та струм через телеметрію. Система постійно показувала фіксоване значення напруги 16В, незалежно від реального, в той час як дані RPM та температури надходили коректно. Проблема діагностувалася при підключенні через UART та з використанням EDT.

Рішення

Для отримання даних про напругу та струм на польотному контролері Matek H743, коли ESC не передає телеметрію, рекомендовано використовувати вбудований ADC. Для цього необхідно встановити `BATT_MONITOR=4` (Analog Voltage and Current). Для систем з високою напругою (напр., 12S), слід підключати живлення через зовнішній BEC або Power Module, який знижує напругу до 5V. Для калібрування струму використовуйте дільник 104, як порадив Вирій, що підтверджено як робоче рішення.
