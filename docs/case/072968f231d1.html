Резонанс на частоті 48 Гц на 7-дюймовому дроні з корисним навантаженням

Case 072968f231d1 · ua_ardupilot

#vibration#resonance#notch filter#INS_GYRO_RATE#payload#tuning#ArduPilot

Проблема

Користувач спостерігає сильний резонанс на частоті 48 Гц на 7-дюймовому дроні з вантажем 1 кг. Пік вібрації чітко видно в логах, і він не характерний для шуму двигунів. Ймовірна причина - вібрація від кріплення корисного навантаження.

Рішення

Рекомендовано перевірити кріплення всіх елементів (підвіс, GPS, антени) для усунення механічної причини вібрації. Для програмного вирішення проблеми порадили налаштувати окремий Notch-фільтр на частоту 48 Гц. Також для кращої діагностики шуму рекомендовано повернути параметр `INS_GYRO_RATE` до стандартного значення 0 (1 kHz).
