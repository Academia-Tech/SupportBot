First Layer Quality Issues with eSUN ABS+ Filament

Case 8533680a0d01 · 3dprinting

#ABS#first layer#troubleshooting#z-offset#heat soaking#bed adhesion#eSUN

Проблема

A user is experiencing poor first layer quality ('blobs' or 'turds') specifically when printing with eSUN ABS+ filament. The issue persists across different slicers and hardware, and only affects ABS+, while PLA, PETG, and ASA print cleanly. The filament is confirmed to be dry and is printed from a dryer.

Рекомендація (не підтверджено)

The user's initial settings for eSUN ABS+ were 260°C nozzle and 105°C bed. Initial troubleshooting suggestions included: 1. Re-checking the Z-offset, as it can be sensitive. 2. Ensuring the printer is properly 'heat soaked' before starting a print to allow the bed, frame, and chamber to reach thermal equilibrium. Further experimentation by the user showed improvement by lowering first layer temperatures (nozzle to 250°C, bed to 100°C) and re-enabling `Z_TILT_ADJUST` in the start g-code. Additionally, it was suggested to use a small amount of Z-hop (e.g., 0.2mm) with the type set to 'Auto'. For this to work effectively, `[gcode_arcs]` must be enabled in the Klipper configuration.
