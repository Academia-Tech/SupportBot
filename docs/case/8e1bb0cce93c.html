Flashing Custom Firmware to NOUS A1T Smart Socket Fails

Case 8e1bb0cce93c · tasmota

#flash#ota#firmware#nous a1t#tasmota-minimal#my_user_config.h#compiling#troubleshooting

Проблема

A user was unable to flash a custom Tasmota binary to a NOUS A1T smart socket. Attempts to upload the file via the web interface resulted in '404 - File not found' or other errors, even though the default OTA path worked. The goal was to add extender functionality.

Рішення

The user solved the problem themselves and shared the solution. The root cause was that the custom firmware binary was too large for a direct OTA update. The solution involved two steps: 1. First, flash the `tasmota-minimal.bin` firmware via the web UI to free up space. 2. Then, flash the custom firmware. To create a smaller custom binary in the first place, the user had to edit the `my_user_config.h` file and deactivate unneeded features/libraries before compiling.
