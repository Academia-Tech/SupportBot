Проблема

Se advierte sobre un próximo cambio en la entidad 'sun' de Home Assistant, específicamente que el sensor `sun_solar_rising` pasará a ser un sensor binario. Esto podría afectar a automatizaciones existentes.

Рекомендація (не підтверджено)

El sensor `sun_solar_rising`, que ya tenía valores true/false, se convertirá en un sensor binario. Este cambio afecta a pocos usuarios, ya que la mayoría utiliza `below horizon / above horizon` para determinar si es de día o de noche. Home Assistant busca 'normalizar' la entidad `sun.sun`.
