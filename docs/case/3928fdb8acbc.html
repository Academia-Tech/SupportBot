Проблема

Un usuario instala la nueva integración oficial de Immich en Home Assistant. Tras introducir la URL del servidor y la API key, la conexión se establece correctamente, pero solo aparecen cuatro entidades relacionadas con el espacio de almacenamiento del servidor, sin ninguna entidad para visualizar las fotos.

Рішення

El propio usuario, tras investigar, descubre que se trata de un bug conocido y ya reportado en la versión inicial de la integración oficial. La funcionalidad para mostrar fotos aún no está implementada o no funciona correctamente. La solución actual es esperar a una futura actualización de la integración que corrija este problema.
