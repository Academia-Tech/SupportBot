Enrutamiento de Tailscale a través de NordVPN en Ubuntu VM

Case 3319764f4b9f · naseros

#VPN#Tailscale#NordVPN#Ubuntu Server#Máquina Virtual#Redes#Enrutamiento

Проблема

El usuario intenta configurar una máquina virtual con Ubuntu Server, Tailscale (como Exit Node) y NordVPN para que todo el tráfico de la red Tailscale salga por la IP de NordVPN. Al conectar NordVPN, Tailscale deja de funcionar y no hay conectividad.

Рішення

Para lograr el enrutamiento de Tailscale a través de NordVPN, se sugiere intentar excluir Tailscale de la VPN de NordVPN (si la aplicación lo permite). Alternativamente, este tipo de configuración es más sencilla de implementar utilizando software de router/firewall como OpenWrt u OPNsense. El usuario ha indicado que investigará estas opciones.
