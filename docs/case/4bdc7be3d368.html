Проблема

Un usuario busca dónde comprar cajas de mecanismos (cajetines) con mayor profundidad para poder instalar dispositivos de domótica como relés o interruptores inteligentes.

Рекомендація (не підтверджено)

Se proponen varias soluciones para alojar módulos domóticos que no caben en cajas de mecanismos estándar: 1. Utilizar cajas de registro amplias en lugar de las de mecanismos. 2. Unir dos cajas estándar en tándem para duplicar la profundidad. 3. Buscar modelos de fabricantes con un "bolsillo" para relés. 4. Si la pared es de ladrillo, picar el fondo para ganar espacio. 5. **Recomendación específica:** El fabricante Legrand ofrece cajas de mecanismos más profundas (hasta 50mm) que son de buena calidad y pueden ser suficientes.
