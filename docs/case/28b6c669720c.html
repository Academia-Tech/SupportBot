Cómo utilizar un router HGU de Movistar con la fibra de Digi

Case 28b6c669720c · naseros

#digi#movistar#hgu#ont#router#fibra#idont#nat loopback#configuración

Проблема

Un usuario desea utilizar un router HGU de Movistar en una conexión de fibra de Digi para conservar la funcionalidad de NAT Loopback. Se pregunta si es posible conectarlo a una ONT externa o si se puede configurar un puerto LAN del HGU como WAN.

Рекомендація (не підтверджено)

Para usar un router propio o de otro operador con Digi, es necesario solicitar a la compañía el IDONT (contraseña GPON), que suelen facilitar. Sin embargo, se perdería el servicio de telefonía fija. Aunque el HGU no tiene puerto WAN RJ45, se discute la posibilidad teórica de reconfigurar un puerto LAN, advirtiendo sobre la complejidad de la configuración de VLANs. Se comparten dos enlaces con guías y discusiones sobre el tema: - https://bandaancha.eu/foros/sustituir-router-ont-digi-router-hgu-1736926 - https://www.frikidelto.com/tutorial/como-conseguir-el-usuario-y-contrasena-pppoe-para-instalar-un-router-neutro/
