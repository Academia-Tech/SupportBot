Проблема

Un usuario pregunta cómo funcionan las máscaras de privacidad en cámaras de videovigilancia, si es una función del software o de la cámara, y si la máscara se puede eliminar posteriormente de la grabación para ver la imagen completa.

Рішення

Ante la duda sobre la legalidad de grabar zonas comunes como un rellano con un videoportero, se recomienda consultar a la AEPD para casos particulares. Como solución técnica para cumplir la ley de protección de datos, se explica el uso de "máscaras de privacidad". Esta función, disponible en software de NVR (como Frigate) o en el firmware de algunas cámaras, permite dibujar áreas (generalmente cuadrados negros) sobre la imagen para ocultar permanentemente zonas sensibles (vía pública, propiedad del vecino). Se confirma que la información de la zona enmascarada no se graba y no se puede recuperar, solucionando así el problema de privacidad.
