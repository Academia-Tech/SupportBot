Controlling Remote Devices with a Multi-Button Tasmota Switch

Case 6fde6cfc5437 · tasmota

#multi-button switch#device groups#rules#remote control#ButtonTopic#Athom

Проблема

A user with a 3-button Athom wall switch wants to use the 2nd and 3rd buttons to control other Tasmota devices. However, when they set a `ButtonTopic`, it applies to all buttons, preventing local control of the first relay.

Рекомендація (не підтверджено)

Two methods were recommended: 1. **Device Groups:** Put the devices into a device group for synchronized control. 2. **Rules:** Use Tasmota rules to trigger actions on remote devices when a specific button is pressed. This allows for more granular control.
