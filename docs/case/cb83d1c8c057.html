Налаштування заборони армінгу дрона при піднятому стіку газу

Case cb83d1c8c057 · ua_ardupilot

#arming#arming check#throttle#safety#configuration#ARMING_RUDDER#PILOT_THR_BHV

Проблема

Початківець в ArduPilot шукає спосіб заборонити армінг дрона, якщо стік газу (throttle) знаходиться не в нульовому положенні, аналогічно до поведінки в Betaflight.

Рішення

Пояснено, що така поведінка є стандартною для режимів Stabilize та Acro завдяки передпольотним перевіркам (Arming Checks). Для армінгу стіками необхідно налаштувати параметр `ARMING_RUDDER`, що також вимагає нульового положення газу. Додатково згадано параметр `PILOT_THR_BHV` для налаштування поведінки газу.
