Проблема

Tras actualizar a un router Movistar WiFi 6 Go y mantener el mismo SSID y contraseña, los dispositivos no se conectan. La sugerencia de 'olvidar la red' no es práctica para 40+ dispositivos, especialmente los IoT de 2.4GHz. El router está en WPA2 y no permite cambiarlo.

Рекомендація (не підтверджено)

Se sugiere que el problema podría estar relacionado con el método de encriptación (WPA3 vs WPA2-PSK), aunque el usuario confirma que está en WPA2 y no puede cambiarlo. También se recomienda probar a desactivar el WiFi AX (WiFi 6) y usar WiFi N, aunque el usuario reporta que esto tampoco resuelve el problema.
