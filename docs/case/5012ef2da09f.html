Проблема

Un usuario está intentando cambiar su coordinador Zigbee actual por un Sonoff Dongle E, pero está encontrando errores y no consigue que funcione.

Рекомендація (не підтверджено)

Se recomienda asegurarse de dos puntos clave para solucionar el problema: 1. Actualizar el firmware del Sonoff Dongle E a la última versión disponible. 2. Verificar que la integración (por ejemplo, Zigbee2MQTT) esté también actualizada y que la configuración sea la correcta para el chipset del Dongle E (EZSP/Ember). Se sugiere buscar una guía específica para este proceso.
