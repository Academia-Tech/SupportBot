Проблема

Un usuario busca domotizar sus persianas con Zigbee. Ya tiene la alimentación para el motor y necesita recomendaciones para un relé y, sobre todo, un pulsador inalámbrico para evitar hacer rozas en la pared.

Рекомендація (не підтверджено)

Se sugieren varias opciones de pulsadores inalámbricos Zigbee que se pueden pegar en la pared: - **Ikea RODRET:** Un regulador con botones claros de subir y bajar. - **Botón inalámbrico de Xiaomi:** Un solo botón que puede configurarse para diferentes acciones (pulsar sube, pulsar para, etc.). - **Pulsadores de AliExpress:** Se menciona que se pueden encontrar fácilmente interruptores con leyendas de subida/bajada en esta plataforma.
