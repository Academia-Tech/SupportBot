Проблема

Un usuario ha montado un homeserver con ZimaBlade y ZimaOS, usando Immich para fotos. No consigue acceder remotamente a Immich a través de Tailscale, aunque el servidor aparece conectado. Pregunta sobre la gestión de acceso remoto del Fritz! 5590 o alternativas a Tailscale.

Рекомендація (не підтверджено)

Se recomienda revisar la configuración de Immich por si hay restricciones de IP que impidan el acceso remoto. Como alternativa a Tailscale, se sugiere utilizar la VPN Wireguard integrada en los routers Fritz!Box para un acceso remoto seguro a toda la red doméstica, evitando depender de servicios de terceros. Se proporciona un enlace al grupo de Telegram de usuarios de Fritz!Box para soporte adicional: `https://t.me/UsuariosAVMyFRITZBOX`.

Hola,

Si puedes no utilizar Tailscale, mejor, ya que estás utilizando la Red de un tercero y hoy puede ser segura y mañana, los hackean, cambian las políticas, etc, así que si puedes, mejor utiliza únicamente tu Red.

Para el acceso remoto, sin depender de terceros, los routers Friztbox, disponen de VPN Wireguard, puedes accta toda la red de casa, incluido el NAS o cualquier otro dispositivo que tengas (impresora, Home Assistant etc), es como si conectases un cable a tu Red desde fuera de casa. No es complicado de configurar y funciona muy rápido.
