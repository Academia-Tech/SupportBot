Проблема

Пошук ефективних рішень для забезпечення відеозв'язку та керування дроном на відстані до 25 км.

Рішення

Для забезпечення дальнього відеозв'язку та керування дроном на відстані до 25 км було запропоновано та обговорено кілька рішень: 1. **Cosmostreamer GD1:** Рекомендовано як ефективне рішення (https://cosmostreamer.com/products/gd1/). Користувач підтвердив, що це гарна ідея, і раніше вже використовував його. Також згадується можливість отримання безкоштовної ліцензії для військових. 2. **Подовжувачі USB:** Для виносу окулярів на щоглу пропонувалися подовжувачі USB по витій парі (наприклад, Digitus DA-70141). Однак, для DJI Goggles 2 потрібен Type-C, а не USB-A. Можливі варіанти: пошук USB-C подовжувача, використання перехідника, або модифікація кабелю/внутрішньої проводки для розділення живлення та даних. 3. **Винос пульта на щоглу з HDMI-to-Ethernet:** Альтернативний підхід полягає у виносі пульта на щоглу, виведенні відео з HDMI пульта, конвертації його в Ethernet (наприклад, за допомогою HDMI extender до 100м, як Cablexpert DEX-HDMI-03) та передачі в бліндаж, де відбувається зворотна конвертація. Живлення пульта на щоглі можна забезпечити від FPV батареї через DC-DC конвертер. Проте, автор питання вважає цей варіант не дуже практичним через необхідність постійного ввімкнення пульта та швидке розрядження батареї. 4. **Одноплатний комп'ютер з Android та DJI Fly:** Запропоновано використовувати одноплатний комп'ютер (наприклад, Raspberry Pi 5) з Android та старою версією DJI Fly як альтернативу Cosmostreamer. Цей варіант працює адекватно лише з Raspberry Pi 5, але потребує мишки для навігації по інтерфейсу, що вимагає додаткової лінії зв'язку на щоглу. Можливе рішення — використання HDMI + USB по одному кабелю. 5. **Комплексна система з Phoenix антеною та Cosmostreamer:** Детально описано робочу систему: антена Phoenix, окуляри з виведеними SMA виходами під щоглою, підключення до антени. Окрема батарея для антени. Окуляри підключаються до Cosmostreamer (з додатковим BEC для живлення окулярів). Оптичний HDMI кабель потрібної довжини йде в підвал, де підключається до цифрової плати відеозахвату, від неї USB в ноутбук (для запису/трансляцій) та HDMI в монітор. Керування йде окремо по витій парі з антени в пульт.
