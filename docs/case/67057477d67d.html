Cómo hacer copias de seguridad de WhatsApp directamente a un NAS

Case 67057477d67d · naseros

#WhatsApp#backup#copia de seguridad#NAS#Android#FolderSync Pro#Synology

Проблема

Un usuario busca una forma de realizar copias de seguridad de WhatsApp desde su teléfono Android directamente a su NAS (Synology), para evitar la lentitud de las copias en Google Drive.

Рішення

Se recomienda utilizar la aplicación para Android 'FolderSync Pro' para realizar copias de seguridad de los archivos multimedia de WhatsApp directamente a un NAS. La aplicación es muy configurable, permitiendo crear tareas de sincronización incremental (solo copia los archivos nuevos o modificados) y programarlas con la frecuencia deseada (incluso de forma casi instantánea). Otro usuario confirma que es una 'muy buena app'. Se aclara que esta solución es para los archivos multimedia (fotos, vídeos, etc.), no para las conversaciones de chat. Para las conversaciones, se debe seguir utilizando la copia de seguridad integrada de WhatsApp en Google Drive. La combinación de ambos métodos ofrece una protección completa.
