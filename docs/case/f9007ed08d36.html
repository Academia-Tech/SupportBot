Migración de Home Assistant en Raspberry Pi 3 por fin de soporte 32-bit

Case f9007ed08d36 · domotica_es

Проблема

Un usuario con Home Assistant en una Raspberry Pi 3 recibe el aviso de que la versión de 32 bits dejará de tener soporte. Pide consejo sobre si migrar a un mini PC o a una Raspberry Pi 4, asumiendo que su hardware es de 32 bits.

Рішення

Se aclara que el hardware de la Raspberry Pi 3 (y modelos posteriores como el A+) es de 64 bits, por lo que no es estrictamente necesario cambiar de equipo. La solución principal es reinstalar el sistema operativo de 64 bits en la RPi y restaurar una copia de seguridad de Home Assistant. Se añade el matiz de que, en el pasado, Home Assistant recomendaba la instalación de 32 bits debido a los recursos limitados de la placa. Como buena práctica para la migración, se aconseja usar una tarjeta SD nueva para la instalación de 64 bits, permitiendo volver a la anterior si la restauración falla.
