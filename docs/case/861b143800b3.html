Проблема

Un usuario busca una solución para controlar dos circuitos perimetrales (A y B) de un robot cortacésped, que funcionan con una señal de radio a través de un cable de cobre. Quiere usar un relé tipo Shelly para poder seleccionar si el robot debe seguir solo el circuito A (por defecto) o una ruta extendida que incluya el circuito B, para luego volver al A.

Рішення

La discusión evoluciona con múltiples propuestas para conmutar entre dos circuitos perimetrales (A y A+B) para una valla de mascotas por radiofrecuencia (PetSafe). Se sugieren soluciones como usar un Shelly con relés optoacoplados o un esquema con dos relés en paralelo. El usuario `user_824c4d244b` proporciona un esquema detallado utilizando un relé DPDT (Doble Polo, Doble Tiro) que permite conmutar la salida del transmisor entre el circuito A y el circuito A+B, manteniendo la continuidad del bucle en ambos casos. Finalmente, el usuario que inició la consulta confirma que ha implementado la solución con éxito utilizando un solo relé DPDT.
