Проблема

El usuario desea domotizar una campana extractora con 3 velocidades y luz separada, integrándola con Home Assistant para automatizaciones basadas en sensores PM2.5 y, en el futuro, con una placa de inducción inteligente. También busca soluciones para olores externos.

Рекомендація (не підтверджено)

Para domotizar la campana, se puede usar un switch o relé de 4 canales. El cableado típico es una entrada de fase, 3 salidas para las velocidades del ventilador y una salida para la luz. Se recomienda usar una crimpadora de faston y hacer fotos antes de tocar nada. Un vídeo útil es: https://youtu.be/xZ9FGMEFIHk. Si se usa un relé sin entradas de switch (SW), no se podrán mantener los botones físicos directamente. Como alternativa, se pueden usar dos relés de 2 canales. Es crucial verificar con un multímetro si cada velocidad del motor se activa con un solo cable o si requiere una combinación de cables. Además, para mejorar la eficacia, se aconseja dejar una ventana un poco abierta para facilitar la circulación de aire. Para problemas de olores de vecinos, se puede instalar una válvula antirretorno, aunque si el problema persiste, puede ser un defecto en el tiro comunitario que debe tratarse en la junta de vecinos.
