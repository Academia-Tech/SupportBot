Вибір та налаштування ESP-модулів для телеметрії ArduPilot

Case a982ffdb23ec · ua_ardupilot

#телеметрія#ESP8266#ESP32#DroneBridge#Mavlink#Yaapu#ArduPilot#бездротовий зв'язок#ExpressLRS

Проблема

Питання щодо використання ESP-модулів (ESP32/ESP8266) для телеметрії ArduPilot, їх сумісності з різними польотними контролерами та альтернативних рішень, таких як DroneBridge, Mavlink backpack та Yaapu скрипт. Обговорюються обмеження та особливості роботи.

Рішення

Для бездротової телеметрії ArduPilot можна використовувати ESP-модулі (ESP8266/ESP32) з відкритим програмним забезпеченням. Рекомендовано ознайомитись з документацією ArduPilot щодо ESP8266 телеметрії: https://ardupilot.org/copter/docs/common-esp8266-telemetry.html. Модулі V3 на ESP32 підтримують DroneBridge, мають роз'єм під зовнішню антену та можуть використовувати ESP NOW для телеметрії на відстань до 1 км. Дронбрідж є кращим варіантом, але не підтримується на ESP8266. Також варто розглянути використання Mavlink backpack. Існує окремий форк ESP з потужним WiFi, але він не продається на AliExpress. Yaapu скрипт також є крутим, особливо для карт.
