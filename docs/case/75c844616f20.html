Recomendaciones de gestor de contraseñas para iOS y Windows

Case 75c844616f20 · naseros

#Gestor de contraseñas#Seguridad#Bitwarden#Vaultwarden#Keepass#1Password#Self-hosting

Проблема

Un usuario busca recomendaciones de un sistema para gestionar contraseñas, ya que usaba Microsoft Authenticator y este va a dejar de funcionar como tal. Necesita una solución compatible con iOS y Windows.

Рішення

Se recomiendan varias opciones: - **Bitwarden / Vaultwarden**: Es la opción más recomendada. Bitwarden es un servicio en la nube (con plan gratuito y de pago), mientras que Vaultwarden es una implementación auto-alojable (self-hosted) del servidor de Bitwarden, que se puede instalar en un NAS o Raspberry Pi. Un usuario confirma que usa la versión gratuita de Bitwarden para dos personas y funciona bien, y planea migrar a Vaultwarden si el servicio gratuito se restringe. - **KeePass**: Es una opción gratuita y offline, donde la base de datos se guarda como un archivo local que se puede sincronizar con servicios en la nube. - **1Password**: Es una opción de pago muy popular. - **Gestor de contraseñas de Apple/Google**: Son opciones integradas en los ecosistemas, pero menos flexibles.
