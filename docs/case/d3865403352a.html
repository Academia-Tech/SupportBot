Особливості калібрування трубки Піто без доступу до GPS

Case d3865403352a · ua_ardupilot

#трубка піто#калібрування#airspeed#gps#ardupilot#без gps

Проблема

Користувач запитує про особливості калібрування трубки Піто без використання GPS, оскільки спроби відкалібрувати її з GPS у певній місцевості ("в Бразилії") були невдалими. Виникає питання, чи можна довіряти мануалу для калібрування без GPS.

Рекомендація (не підтверджено)

Для калібрування датчика повітряної швидкості (трубки Піто) без GPS обговорювалося кілька методів. Стандартний метод автокалібрування в польоті (`ARSPD_AUTOCAL`) вимагає наявності GPS. Наземне калібрування є складним, оскільки параметр `ARSPD_RATIO` компенсує не тільки сам сенсор, а й його положення. Також важливо мати кришку для трубки Піто та лейбу 'Remove before flight', оскільки ArduPilot калібрує нуль Піто при кожному старті, і набігаючі потоки повітря на землі можуть збивати цю калібровку.
