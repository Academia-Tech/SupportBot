Дрон плаває по висоті: опускається та набирає висоту циклічно

Case e7e66cab3352 · ua_ardupilot

#висота#плавання#барометр#PID#налаштування#ArduPilot#лог

Проблема

Дрон стабільно зависає, але через кілька секунд починає опускатися, потім знову набирає висоту, і так по колу. Користувач шукає причину такої поведінки.

Рекомендація (не підтверджено)

Для плавань по висоті перевірити `mot_thst_hover`, `psc_accz_p`, `psc_accz_i` та інші параметри. Перед налаштуванням PID/інших параметрів обов'язково виконати `MOT_HOVER_LEARN`. Якщо коптер ривками газує трохи вгору-вниз, це може бути ознакою того, що `MOT_THST_HOVER` не відповідає реальному значенню. Для подальшого налаштування PID та фільтрів після автотюну, особливо для конфігурацій зі StabX та сервоприводами, можна звернутися до досвідчених користувачів, які доналаштовували ці параметри вручну для кращого позиціонування.
