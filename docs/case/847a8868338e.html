Проблема

Se plantea la dificultad de recuperar discos virtuales de gran tamaño si el sistema de virtualización (Proxmox) falla, y cómo gestionar las copias de seguridad de terabytes de datos.

Рекомендація (не підтверджено)

Se discute la importancia de tener backups y se menciona que Proxmox puede gestionar todo, con TrueNAS para servicios NAS. Para protección contra fallos de disco sin necesidad de duplicar el espacio con backups completos debido al gran volumen de información (ej. 20TB+ de multimedia), se recomienda el uso de Snapraid. Snapraid ofrece protección contra el fallo de uno o más discos sin requerir una copia completa de los datos, lo que es ideal para colecciones grandes donde la duplicación total es costosa. Un usuario confirmó haber recuperado exitosamente todos sus datos tras el fallo de un disco gracias a Snapraid. Se enfatiza que, aunque no se haga un backup completo, es crucial tener alguna forma de protección contra fallos de hardware, ya que hay contenido que puede ser irrecuperable o muy tedioso de volver a descargar.
