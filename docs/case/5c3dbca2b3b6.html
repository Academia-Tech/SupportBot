Проблема

Un usuario pregunta si es posible controlar dos persianas, que están juntas en el salón, con un único relé. También consulta sobre la existencia de interruptores dobles que quepan en un solo cajillo y si es un error no instalar pulsadores físicos, confiando solo en un mando RF.

Рішення

Se confirma que es posible controlar dos motores de persiana con un único dispositivo como el Sonoff 4CH. Sin embargo, se genera un intenso debate sobre su idoneidad y seguridad. Varios usuarios desaconsejan esta práctica por el riesgo de 'catástrofes' si no se configura correctamente el enclavamiento entre relés para evitar que los motores de subida y bajada se activen a la vez. Para la configuración, se recomienda flashear el dispositivo con Tasmota o ESPHome en lugar de usar el firmware de serie (eWeLink). Con Tasmota, se sugiere una configuración específica: usar un relé para la activación y otro para el sentido (subida/bajada), con `Switchmode 1` o `2`. Se aclara que este método no permite un control porcentual de la posición de forma nativa, ya que el relé no sabe cuándo parar. Para lograrlo, es necesario usar una configuración de 'time based cover' en ESPHome o Tasmota, que calcula la posición basándose en el tiempo de recorrido, aunque se considera una solución 'lejos de ser ideal'. Se advierte que el esquema propuesto por Sonoff es peligroso para usuarios con pocos conocimientos.
