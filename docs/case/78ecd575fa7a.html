Проблема

Un usuario busca una solución para abrir la puerta de una empresa desde un pulsador inalámbrico en el escritorio, evitando el cableado y la necesidad de levantarse constantemente. También pregunta si existen telefonillos con esta función integrada.

Рекомендація (не підтверджено)

Se sugieren varias opciones para añadir un pulsador inalámbrico a un telefonillo y poder abrir la puerta sin cables: 1. **Relé + Botón Inalámbrico**: Instalar un relé (por ejemplo, RF 433) en el telefonillo y emparejarlo con un pulsador inalámbrico. Es crucial verificar primero el modelo del telefonillo y el espacio disponible. 2. **Ring Intercom**: Una solución comercial que permite abrir la puerta y hablar a través del móvil. Es integrable con Home Assistant y se puede controlar por voz con Alexa. Se proporciona un enlace para verificar la compatibilidad del telefonillo: https://ring.com/intercom-compatibility-checker/ 3. **Mecanismo de Portero Automático**: Como alternativa, se puede instalar un mecanismo de cierre eléctrico como el de Leroy Merlin (https://www.leroymerlin.es/productos/cierre-electrico-cerradura-desbloqueo-automatico-y-corriente-alterna-11954663.html), que se puede accionar de diversas maneras. Se aconseja encarecidamente investigar primero el modelo del telefonillo y la instalación actual antes de comprar cualquier dispositivo.
