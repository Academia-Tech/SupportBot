How to prevent spammers from messaging members of a Telegram group

Case cde6a59dde8d · tasmota

#telegram#spam#privacy#group settings#member list#administration#security#community management

Проблема

Users reported receiving unsolicited private messages about financial investments from other members of the group (e.g., "Tommaso De Luca", "yasha Klein"). The issue was attributed to the group's member list being public, which allows spammers to easily scrape contacts and send direct messages.

Рішення

A user suggested that the spam could be significantly reduced by hiding the group's member list, a feature in Telegram's group settings. Another user then tagged the administrators to request this change. An admin, user_70d4e26e86, confirmed they had taken action to hide the list to see if it would help mitigate the spam problem. This is a key privacy setting for large public groups to prevent member harvesting.
