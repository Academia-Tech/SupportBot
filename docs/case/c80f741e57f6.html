Сильна нестабільність та осциляції 10-дюймового дрона при спробі зльоту

Case c80f741e57f6 · ua_ardupilot

#instability#oscillation#10-inch#tuning#filtering#harmonic notch#bidirectional dshot#param file

Проблема

Користувач демонструє на відео, як 10-дюймовий дрон на ArduPilot стає абсолютно некерованим одразу після зльоту. Апарат сильно осцилює, хаотично обертається і не може набрати висоту, що свідчить про серйозні проблеми з налаштуваннями PID-регулятора або фільтрації.

Рекомендація (не підтверджено)

Надано вичерпний набір параметрів для налаштування фільтрації з нуля, оскільки проблема, ймовірно, не в PID-ах (стандартні підходять для 10"), а у відсутності фільтрів. Рекомендовано налаштувати Bi-directional DShot та гармонійний notch-фільтр на основі телеметрії ESC. Надано готовий список параметрів для завантаження через файл `.param`, що включає: INS_ACCEL_FILTER,10 INS_GYRO_FILTER,50 INS_HNTCH_ENABLE,1 та пов'язані параметри (MODE=3, HMNCS=7, OPTS=38 і т.д.) SERVO_BLH_AUTO,1 SERVO_DSHOT_ESC,1 або 2 Також надано посилання на документацію та пораду щодо налаштування MOT_HOVER_LEARN після першого польоту.
