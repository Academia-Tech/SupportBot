Проблема

Un usuario quiere reemplazar el router de su proveedor O2 por una ONT externa (modelo EG8120L) y un sistema WiFi mesh. Pregunta sobre la viabilidad, los pasos y la configuración necesaria para que funcione el servicio de internet y telefonía.

Рекомендація (не підтверджено)

Para sustituir el router HGU de O2/Movistar por una ONT externa (ej. Huawei EG8120L) y un router neutro, es imprescindible obtener la credencial de la fibra, conocida como Contraseña GPON o PLOAM Password. Esta clave autentica la ONT en la red del operador. **Pasos para la configuración:** 1. **Obtener la Contraseña GPON:** Se extrae del HGU original del operador, accediendo a su interfaz web (normalmente 192.168.1.1). A veces es necesario desconectar el cable de fibra para que la muestre en la sección de estado o mantenimiento GPON. 2. **Configurar la ONT externa:** Introducir la Contraseña GPON obtenida en la interfaz de configuración de la nueva ONT para que sincronice con la OLT del operador. 3. **Configurar el Router Neutro:** Se deben configurar las VLANs correspondientes: * **VLAN ID 6:** Para Internet, usando PPPoE con usuario `adslppp@telefonicanetpa` y contraseña `adslppp`. * **VLAN ID 3:** Para telefonía VoIP (si se tiene contratada). Se advierte que O2 no da soporte a esta configuración. Sí, si deseas utilizar una ONT externa con O2 (o Movistar, ya que comparten la misma infraestructura de fibra), sí se requiere una credencial específica para la ONT. Esta credencial se conoce comúnmente como ID ONT, Contraseña GPON o PLOAM Password.
¿Por qué se necesita esta credencial?
Cuando tu fibra óptica llega a tu casa, la OLT (Optical Line Terminal) del operador en la central necesita identificar y autenticar tu ONT para permitirle el acceso a la red. Esta autenticación se realiza mediante esta credencial única. Sin ella, tu ONT externa no podrá sincronizarse correctamente con la red de O2 y no tendrás servicio.
¿Cómo se obtiene esta credencial?
La forma más común de obtener la Contraseña GPON es extrayéndola del HGU (Home Gateway Unit) que te proporciona O2. El HGU ya tiene esta información programada.
El proceso para extraerla suele ser el siguiente:
* Acceder al HGU: Necesitarás acceder a la interfaz de configuración web de tu router HGU de O2 (normalmente en 192.168.1.1).
* Buscar la sección de GPON/ONT: Dentro de la configuración, suele haber una sección dedicada a la fibra óptica o GPON, a veces bajo menús como "Mantenimiento" o "Estado".
* Localizar la contraseña: En algunos modelos, es posible que necesites desconectar el cable de fibra del HGU para que te muestre la opción o el campo donde se visualiza el ID ONT o la contraseña GPON. Es una clave alfanumérica, a menudo de 10 o 20 caracteres.
Importante: Este proceso no siempre es oficial ni soportado por O2. Se considera una "modificación" de la configuración que te proporcionan, y O2 siempre recomendará usar su HGU. Sin embargo, es una práctica común entre usuarios avanzados que desean utilizar su propio equipo.
¿Qué se hace con la credencial una vez obtenida?
Una vez que tengas el ID ONT o Contraseña GPON, deberás introducirla en la configuración de tu ONT externa. El proceso exacto dependerá del modelo y fabricante de tu ONT. Generalmente, accedes a la interfaz web de la ONT y buscas una sección de "GPON", "WAN" o "System" donde puedas introducir esta credencial para que se sincronice con la red del operador.
Consideraciones adicionales:
* VLANs: Además de la contraseña GPON de la ONT, si usas un router neutro después de tu ONT externa, necesitarás configurar las VLANs de O2/Movistar:
* VLAN ID 6: Para el servicio de Internet (PPPoE con usuario adslppp@telefonicanetpa y contraseña adslppp).
* VLAN ID 3: Para el servicio de VoIP (teléfono fijo), si lo tienes contratado. Esto es más complejo y a menudo requiere un adaptador ATA SIP y configurar credenciales SIP específicas.
* Soporte: O2 no ofrece soporte para configuraciones con ONTs externas o routers neutros, ya que su servicio está diseñado para funcionar con el HGU que proporcionan. Cualquier problema que surja de esta configuración será tu responsabilidad.
En resumen, sí, la ONT externa requiere una credencial (ID ONT/Contraseña GPON) para conectarse a la red de O2, y esta debe extraerse del HGU suministrado por el operador.
