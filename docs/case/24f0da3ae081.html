Configuring Ethernet on M5Stack PoEsp32 with Custom Tasmota Firmware

Case 24f0da3ae081 · tasmota

#ethernet#esp32#m5stack#poe#custom firmware#user_config_override#networking#troubleshooting

Проблема

A user is unable to get the Ethernet interface working on an M5Stack PoEsp32 after flashing a custom-compiled Tasmota firmware. The provided serial logs show Tasmota starting in Wi-Fi AP mode, and the physical Ethernet port LEDs are not active. The user shared their `user_config_override.h` for diagnosis.

Рекомендація (не підтверджено)

Several troubleshooting steps were suggested to enable Ethernet and bypass the default Wi-Fi setup: 1. Use the `EthIPAddress` command via the serial console to manually configure the IP settings. Documentation is available at: https://tasmota.github.io/docs/Commands/#ethernet 2. Disable the Wi-Fi manager at startup using a rule, for example: `Rule1 ON system#boot DO wifi 0 ENDON`. 3. A link to a relevant GitHub discussion on the topic was also shared: https://github.com/arendst/Tasmota/discussions/17794
