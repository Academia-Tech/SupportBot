Проблема

Обговорюються різні способи калібрування датчика струму для отримання точних показників споживання. Прості методи, такі як порівняння ємності, вважаються неточними. Потрібні надійніші способи вимірювання реального струму, особливо для ESC 4-в-1, де важко підключити зовнішній шунт.

Рекомендація (не підтверджено)

Надано декілька методів калібрування датчика струму, від простих до точних: 1. **Струмові кліщі:** Найпростіший метод для ESC 4-в-1. Використовуються струмові кліщі для постійного струму, якими обхоплюється силовий дріт. 2. **Зовнішній ватметр:** Використовувати зовнішній ватметр (на фото IMG:0 показаний синій прилад Watt Meter 0-60V 0-100A), підключивши його в розрив живлення. 3. **Регульоване навантаження:** Найточніший метод. Використовувати регульоване електронне навантаження і калібрувати датчик на струмі, що відповідає крейсерському режиму польоту. 4. **Телеметрія з ESC:** Використовувати дані про струм, що передаються з регуляторів по цифровому протоколу (DShot). Цей метод не враховує споживання периферії. Також наголошується на важливості одночасного калібрування датчика напруги.
