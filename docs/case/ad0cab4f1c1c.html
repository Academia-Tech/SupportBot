How to use a PIR sensor with Tasmota and Home Assistant without a physical relay

Case ad0cab4f1c1c · tasmota

#Tasmota#Home Assistant#PIR Sensor#Switch#Relay#Decouple#SetOption114#MQTT

Проблема

A user wanted to connect sensors like PIRs to a Tasmota device's inputs (configured as switches) and have their status reflected in Home Assistant. However, they wanted to do this without the input being tied to controlling a physical relay, in order to use the device for sensor aggregation only. The user was struggling to explain this due to using a translator.

Рішення

The user discovered the solution themselves after a lengthy discussion. The `SetOption114 1` command is used to decouple switches from relays. By setting this option, the switch input will report its state via MQTT (which Home Assistant can see) but will not trigger the corresponding relay. This allows a PIR sensor connected to a switch input to act as a standalone sensor for home automation.
