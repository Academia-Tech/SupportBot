Проблема

Un usuario necesita reemplazar el router de su operadora y los puntos de acceso en una casa grande de 200m². Busca una solución personalizable que permita crear varias redes y segmentación, y que sea adecuada para una red con muchos dispositivos de domótica.

Рекомендація (не підтверджено)

Se amplía la recomendación unánime de usar equipos Ubiquiti UniFi, desaconsejando firmemente el router del operador para domótica. La conversación confirma la idoneidad de la solución y el usuario se decide por ella. Se detallan y comparan varias configuraciones y alternativas: 1. **Equipos UniFi recomendados**: UDM-Pro o UDR7 + Switch PoE + Puntos de Acceso (AP) U6+. También se menciona el UDM base como opción. 2. **Alternativas**: Se discuten TP-Link Omada (EAP650) y Deco (XE75), pero se concluye que la experiencia de usuario y el software de UniFi son superiores. 3. **Consejos clave**: Se aconseja que todos los componentes de red sean de la misma marca para una mejor experiencia y se confirma que los AP de UniFi soportan configuración en malla (mesh). El usuario finalmente se convence de usar la pila de Ubiquiti, planeando añadir más APs si la cobertura inicial no es suficiente.
