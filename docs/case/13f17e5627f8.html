Recomendaciones de conexión a internet para casa de pueblo sin fibra óptica

Проблема

Un usuario necesita una solución de internet para una casa de pueblo aislada donde no llega la fibra, pero sí hay buena cobertura 5G. La conexión se usaría de forma esporádica (fines de semana, vacaciones), por lo que busca la opción más adecuada para este tipo de uso.

Рекомендація (не підтверджено)

Se discuten dos alternativas principales para dotar de internet a la vivienda: 1. **Router 5G con tarjeta SIM:** Es la opción más flexible y recomendada para uso no continuo. Se mencionan planes como "Ilimitodo" de Digi (por 12-25€/mes) o una línea de datos auxiliar de O2 (150GB por 15€). En cuanto al hardware, se sugiere el router Zowee Flybox 5G, que se puede adquirir a buen precio siendo cliente de Simyo. 2. **Starlink:** Se considera la mejor opción en términos de rendimiento y velocidad si no hay fibra. Se comenta que hay ofertas sin coste de equipo, pero conllevan un compromiso de permanencia de 12 meses, lo que puede no ser ideal para una segunda residencia. Además, el coste mensual es considerablemente más alto que una tarifa de datos 5G.
