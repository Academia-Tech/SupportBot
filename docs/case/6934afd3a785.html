Проблема

Користувач зіткнувся з проблемою сильних вібрацій та перегріву моторів на 10-дюймовому квадрокоптері під час зльоту. Були перевірені рама, мотори, пропелери, стек, але проблема залишається. Дефолтні PID та спроби увімкнення нотч-фільтрів не допомогли.

Рекомендація (не підтверджено)

Для діагностики та вирішення проблеми користувач Адмін Andrii 🇺🇦 (user_3cc6b8d440) надав наступні рекомендації: 1. **Перевірка телеметрії RPM:** Переконатися, що телеметрія RPM надходить з ESC. Для цього в Mission Planner у вкладці 'Дата' та 'Статус' перевірити наявність ESC RPM після запуску моторів. 2. **Налаштування фільтрів:** Якщо телеметрія RPM є, налаштувати фільтри. Рекомендовано використовувати динамічний нотч-фільтр. 3. **Перевірка PID:** Якщо після налаштування фільтрів проблема залишається, перевірити PID. Для P та I значення 0.135 є прийнятним, але для D значення має бути на порядок менше. 4. **Аналіз логів:** Якщо проблема не вирішується, надати логи для детального аналізу. Команда підтримки Dmytro Radomtsev (user_2c254dce42) додав, що значення 0.135 є типовим для P та I, але для D потрібно значно менше значення.
