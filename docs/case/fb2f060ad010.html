Fallo general de las integraciones de Alexa en Home Assistant

Case fb2f060ad010 · domotica_es

#alexa#alexa media player#home assistant#integración#api#hacs#bug#solucionado

Проблема

Varios usuarios reportan un fallo simultáneo tanto en la integración de HACS "Alexa Media Player" como en la integración oficial de Alexa en Home Assistant. Los dispositivos aparecen como no disponibles y no se pueden controlar. El problema parece estar relacionado con la API de Amazon en ciertos países, como España.

Рішення

Se identificaron dos soluciones que resolvieron el problema: 1. **Solución temporal**: Cambiar el dominio de Amazon en la configuración de la integración a un país que no esté afectado, por ejemplo, Italia (`alexa.amazon.it`). Esto permite restaurar la conexión de forma inmediata. 2. **Solución definitiva**: Actualizar la integración "Alexa Media Player" a través de HACS a la versión que corrige el error (v5.7.3). Varios usuarios confirmaron que tras aplicar esta actualización, la integración volvía a funcionar correctamente.
