A user asked about integrating PV micro-inverters for energy monitoring. The conversation evolved into a detailed explanation of how to build a DIY, non-invasive AC current monitor using an ESP8266 and a current clamp sensor.

Рекомендація (не підтверджено)

A detailed DIY guide was provided for building an AC current monitor. The method uses a non-invasive AC current clamp (a split-core current transformer). The AC output from the clamp is rectified to DC using a self-made full-wave bridge rectifier built with four Schottky diodes and a small smoothing capacitor. The resulting DC voltage is then fed into the analog input of an ESP8266 for measurement. The guide includes details on component selection, circuit design, and calibration within Tasmota. It has been confirmed that a Non-Invasive Current Sensor like the SCT-013 100A is compatible with this DIY setup.
