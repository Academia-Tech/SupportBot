Проблема

Un usuario busca recomendaciones de sistemas operativos para su HP Proliant Microserver Gen8, ya que ZimaOS no es compatible debido a problemas con UEFI.

Рекомендація (не підтверджено)

Se recomiendan varios sistemas operativos: TrueNAS, Unraid, Debian, Proxmox, Windows Server 2008 R2 o ESXi 5.5. Se advierte que instalar Linux 'palo seco' puede ser difícil para usuarios sin experiencia. Un usuario comparte su configuración exitosa en el mismo hardware: Debian como base, con CasaOS instalado encima y gestionando un RAID 6 de 4 discos a través de Cockpit.
