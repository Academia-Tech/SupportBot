Проблема

Пошук рішень для реалізації керування дроном та передачі відеосигналу через супутниковий інтернет Starlink.

Рішення

Для керування та відеозв'язку через Starlink можна використовувати одноплатний комп'ютер (SBC). Керування (UART) пакується через `ser2net`, а відеопотік з IP-камери або AV-конвертера передається через RTSP сервер. Ця схема вже активно використовується для бойових дронів. Її ключові переваги: підвищена безпека пілотів, відсутність наземного обладнання, стійкість до гармонік радіопередавачів. Також згадується, що R2D2 має готові рішення для ретрансляторів на Starlink. Існують проекти, де Starlink залишається на землі, а ретранслятор використовує WiFi HaLow (700-900МГц, 200мВт, 4-8МГц канал), який менш чутливий до гармонік. Для відеозв'язку також практикується розрахунок гармонік від частот керування та призначення відповідних відеоканалів.
