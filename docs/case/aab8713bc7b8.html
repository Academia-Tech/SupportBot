Problemas al flashear ESPHome en un Sonoff Basic R2 v1

Case aab8713bc7b8 · domotica_es

#ESPHome#Tasmota#Sonoff#Sonoff Basic#flasheo#firmware#ESP8266

Проблема

Un usuario informa que ha "brickeado" dos dispositivos Sonoff Basic R2 v1 al intentar flashear ESPHome directamente desde la herramienta espbuilder de Home Assistant y pregunta si este modelo es compatible.

Рекомендація (не підтверджено)

Se explica que, aunque la mayoría de los Sonoff son compatibles, la versión v1 podría ser una excepción. Se recomienda un procedimiento más seguro y detallado: 1. Primero, flashear la última versión de Tasmota en el dispositivo. 2. Una vez con Tasmota, migrar a ESPHome desde la propia interfaz web de Tasmota, siguiendo la guía oficial: https://esphome.io/guides/migrate_sonoff_tasmota 3. **Importante:** Para los dispositivos con chip ESP8266, como el Sonoff Basic, es crucial utilizar el archivo de firmware comprimido (`.bin.gz`) durante el proceso de migración. 4. Si el dispositivo queda inaccesible ('tostado'), se puede intentar recuperarlo flasheando un firmware básico de ESPHome que incluya `captive_portal` y modo `ap` para forzar la conexión.
