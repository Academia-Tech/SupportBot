How to Trigger a Rule's Action Immediately Upon Activation

Case cd6d75ccf608 · tasmota

#rules#trigger#once#automation#rule activation

Проблема

A user wants to trigger the action part of a Tasmota rule (e.g., `rule2`) immediately when it is enabled from another rule (e.g., `rule1`), without waiting for a specific event trigger within `rule2`.

Рішення

To execute a rule for a single run immediately, use the `once` keyword. For example, from within `rule1`, you can issue the command `Rule2 once`. This will activate `rule2`, execute its `DO` commands one time, and then disable it again. The user confirmed this solved their problem.
