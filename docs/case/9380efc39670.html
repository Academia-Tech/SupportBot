Налаштування порядку моторів та доступу до конфігуратора AM32 ESC

Case 9380efc39670 · ua_ardupilot

#am32#esc#servo output#motor order#frame_type#servo_blh_auto#конфігуратор#вирій про

Проблема

Користувач зіткнувся з двома проблемами на квадрокоптері з ESC 4-в-1 (Вирій Про 15): 1. Неправильний порядок та/або напрямок обертання моторів при різних значеннях FRAME_TYPE. 2. Неможливість підключитися до конфігуратора AM32 для зміни налаштувань ESC.

Рішення

Оригінальні поради (підсумок попередніх рішень): встановити правильний порядок моторів у Servo Output (наприклад 1,2,3,4) та за потреби в Full Parameter List увімкнути автонастройку SERVO_BLH_AUTO/інші параметри доступу до ESC. Додано (оновлення): практичні кроки діагностики та виправлення непередбачених проблем з моторами. Важливо розрізняти порядок розміщення моторів та напрям їх обертів. Для діагностики рекомендовано намалювати схему, зайти в мотор-тест і по черзі запускати мотори, відмічаючи результат. Реверсні пропелери (reversed props) встановлюють для того, щоб відштовхувати гілки від дрона, а також для зменшення забруднення камери, хоча є думка, що польотний контролер все одно забруднюється. На зображеннях показано класичну схему QUAD X та реверсну схему QUAD X (Betaflight) з відповідними напрямками обертання пропелерів.
