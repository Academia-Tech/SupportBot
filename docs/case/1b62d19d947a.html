Sustituir router Eero de Adamo por un TP-Link AX53

Case 1b62d19d947a · naseros

#adamo#eero#tp-link#ax53#doble nat#ont#vlan#sustituir router

Проблема

Un usuario con fibra de Adamo, que utiliza una ONT independiente y un router Eero, añade un segundo router TP-Link AX53, lo que le genera una configuración de doble NAT. Busca la forma de eliminar el router Eero y usar solo el TP-Link, pero no sabe cómo configurarlo.

Рішення

La solución final y confirmada fue conectar el router TP-Link AX53 directamente a la ONT de Adamo, eliminando por completo el router Eero. Se indica que para Adamo, la configuración necesaria es: Tipo de conexión DHCP y VLAN de Internet "603" con prioridad "1". Sin embargo, el usuario reporta que su router TP-Link se autoconfiguró y obtuvo conexión inmediatamente sin necesidad de introducir estos parámetros manualmente.
