Проблема

A user with an Elegoo Neptune 3 Pro is experiencing very poor print quality, characterized by significant layer inconsistencies (Z-wobble), a peeling first layer, and other artifacts like blobs and a prominent vertical seam. The user has tried basic recalibration without success.

Рекомендація (не підтверджено)

A comprehensive list of troubleshooting steps was provided by the community. 1. **Mechanical Checks:** The primary suspect is Z-axis wobble, potentially from a bent leadscrew. Other mechanical checks should include looking for slack belts, wobbly wheels on the carriages, and loose stepper motors or bearings. 2. **Full Recalibration:** The user performed a full recalibration (E-steps, PID tuning, speed reduction, Z-axis greasing) which resulted in only minor improvements. 3. **Slicer & Firmware Settings:** It was suggested to check the g-code preview for anomalies and to tune Pressure Advance if using Klipper firmware. A user also noted that the infill pattern was visible on top surfaces, pointing to a potential slicer setting issue. 4. **Extruder Gear Alignment:** A key finding was a video (https://www.youtube.com/watch?v=c6JmCdovE0U) demonstrating that misaligned gears in direct drive extruders can cause repeating surface defects. This led to the recommendation to inspect the extruder's gear alignment and consider upgrading the gear set (e.g., with a RIGDA kit).
