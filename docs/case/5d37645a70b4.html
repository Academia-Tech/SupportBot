Вибір типу карти в Mission Planner після оновлення

Case 5d37645a70b4 · ua_ardupilot

#Mission Planner#карта#інтерфейс#Flight Planner#відображення#навігація#відео

Проблема

У Mission Planner відсутній прямий вибір типу карти в основному інтерфейсі, що ускладнює зміну відображення. Користувач демонструє, як знайти цю опцію.

Рішення

Для зміни типу карти в Mission Planner необхідно перейти в режим 'Flight Planner' (через контекстне меню на карті), де з'явиться відповідний випадаючий список для вибору карт. Рішення було підтверджено як дуже інформативне.
