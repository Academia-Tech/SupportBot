Проблема

Un usuario busca un router que acepte tarjeta SIM 5G para usar en una autocaravana, con un presupuesto ideal de menos de 100€. No encuentra opciones asequibles.

Рішення

Se concluye que un router 5G por menos de 100€ es muy difícil de encontrar. Se ofrecen varias alternativas y recomendaciones: - **Routers 5G recomendados (más caros)**: Se mencionan modelos como TP-Link NX210 (aprox. 200€) o Cudy, que además permiten antenas externas. También se sugiere el Zowee Flybox 5G disponible en la tienda de Simyo por 108€. - **Router 4G+ (Cat 6)**: Varios usuarios recomiendan optar por un buen router 4G+ (Categoría 6) en lugar de un 5G de gama baja. Específicamente, se destaca el modelo **TP-Link MR600** por su buen rendimiento, con un precio de unos 70€ nuevo o 40€ de segunda mano. - **Móvil 5G dedicado**: Usar un smartphone 5G económico exclusivamente para compartir internet en la autocaravana.
