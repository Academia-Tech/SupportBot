Проблема

Después de actualizar el Core de Home Assistant, la integración personalizada de Tapo deja de funcionar, mostrando un error de "Requirements for tapo not found: ['plugp100==5.1.4']". El problema afecta a los usuarios de dispositivos Tapo P100.

Рекомендація (не підтверджено)

El problema es un fallo conocido causado por la actualización del Core de HA que afecta a la dependencia `plugp100`, impactando a dispositivos como el `tapo-p100`. Varios usuarios confirman el error tras actualizar. Se ha reportado que ya existe una solución aportada, aunque no sea la ideal. Se recomienda buscar el fallo en el repositorio de la integración para encontrar la información más reciente y notificarlo para que los desarrolladores puedan solucionarlo.
