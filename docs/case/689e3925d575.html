Troubleshooting Artifacts ('Barcode') on ILI9341 TFT Display

Case 689e3925d575 · tasmota

#tasmota#display#tft#ili9341#esp32#artifacts#capacitor#decoupling#hardware

Проблема

A user connected a 2.8-inch ILI9341 TFT display to an ESP32 running Tasmota. The display was functional but showed an unwanted bar with artifacts, described as a 'barcode'. The user asked for a way to remove this visual glitch.

Рекомендація (не підтверджено)

A recommendation was given to first test the display independently of Tasmota to isolate the issue. To fix display errors and artifacts caused by unstable power, it was suggested to add decoupling capacitors to the display's power terminals: a 10uF capacitor and a 0.1uF capacitor are recommended to stabilize the power supply.
