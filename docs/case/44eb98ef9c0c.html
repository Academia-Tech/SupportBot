How to Decouple the Physical Switch from the Relay on a Sonoff ZBMINI

Case 44eb98ef9c0c · tasmota

#tasmota#zigbee#sonoff#zbmini#decouple#switch#setoption73#zbbp

Проблема

A user wants to decouple the physical switch from the internal relay on a Sonoff ZBMINI (a Zigbee device) which is connected to a Tasmota Zigbee Bridge (ZBBP). They are looking for a command similar to `SetOption73` which is used for Wi-Fi Sonoff MINI devices.

Рішення

It was clarified that the Sonoff ZBMINI is a Zigbee device and does not run Tasmota firmware itself. Therefore, Tasmota commands like `SetOption73` or `SetOption114` do not apply. The decoupling functionality, if available, must be configured on the end device (the ZBMINI) itself, likely through specific Zigbee commands sent from the coordinator (the Tasmota bridge). It is not a standard Tasmota setting that can be changed with a simple command, and the user was advised that any such change would need to be done at the unit level.
