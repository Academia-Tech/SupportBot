Що означає оператор `>>=` в C++ для скриптів Tasmota

Case 034399676cce · tasmota

#scripting#operators#>>=#c++#bitwise#syntax

Проблема

Користувач запитав про значення оператора `>>=` у коді.

Рішення

Оператор `>>=` — це оператор побітового зсуву вправо з присвоєнням. Вираз `C >>= 2` є скороченням для `C = C >> 2`, що означає зсув бітів змінної `C` на дві позиції вправо.
