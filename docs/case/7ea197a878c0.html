NodeMCU with Tasmota Cannot Toggle Off a 12V Relay Module

Case 7ea197a878c0 · tasmota

#nodemcu#relay#tasmota#gpio#logic level#3.3v#5v#troubleshooting

Проблема

A user connected a relay module to a NodeMCU (ESP8266) on pin D4. The relay turns on at boot but cannot be toggled off via the Tasmota web UI. The relay module in question is a 12V model with an optocoupler.

Рекомендація (не підтверджено)

The user reported that a relay connected to a NodeMCU would turn on but not off. After providing a link to the relay module (https://www.electronicscomp.com/1-channel-12v-relay-module-with-optocoupler) and later the specific part number `JQC-3FC(T73)`, it was confirmed that the module requires a trigger voltage higher than the 3.3V provided by the NodeMCU's GPIO pins. The solution would be to use a level shifter, a transistor driver circuit, or a different relay compatible with 3.3V logic.
