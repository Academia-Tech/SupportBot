Móviles Xiaomi se conectan a red Mesh pero no transfieren datos

Case 9521b3ca0ba4 · naseros

#Xiaomi#WiFi Mesh#Conflicto IP#DHCP#IP Fija#Conectividad#Red Doméstica

Проблема

Un usuario experimenta problemas esporádicos con móviles Xiaomi en una red WiFi mesh. Los dispositivos se conectan, obtienen una dirección IP, pero no tienen conectividad a internet hasta que se desactiva y reactiva el WiFi en el móvil. La red mesh tiene un único SSID con band steering.

Рішення

El problema probablemente se debe a un conflicto de direcciones IP en la red local. Esto puede ocurrir si un dispositivo tiene una IP fija que se encuentra dentro del rango de asignación dinámica del DHCP del router. La solución consiste en organizar la asignación de IPs: 1. Reservar IPs para dispositivos específicos a través de su dirección MAC en la configuración del DHCP. 2. Asignar IPs fijas manualmente solo en rangos que estén fuera del pool de DHCP (por ejemplo, si el DHCP asigna de .20 a .150, usar IPs fijas de .2 a .19 o de .151 a .254).
