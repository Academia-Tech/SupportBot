Перегоряють конденсатори на ESC через сильні вібрації та некоректні PID

Case 3f6ec4332855 · ua_ardupilot

#ESC#capacitor#vibration#PID tuning#D-term#filters#BDshot#hardware failure#motors

Проблема

У користувача на дроні з моторами 4315 700KV та 13-дюймовими пропелерами постійно перегорають ніжки конденсаторів на ESC. Проблема виникає під час перших спроб зльоту для налаштування, особливо при перемиканні в режим AltHold, що супроводжується падінням дрона.

Рекомендація (не підтверджено)

Проблема, ймовірно, викликана сильними вібраціями через некоректні налаштування PID (зависока D-складова) та фільтрів для конфігурації з високообертовими моторами. Рекомендовано: 1. Перевірити та налаштувати фільтри. Для першого зльоту можна грубо встановити фільтр низьких частот від 40 Гц. 2. Налаштувати BDshot фільтрацію для ефективного придушення вібрацій від моторів. 3. Вимкнути активне гальмування (brake) в налаштуваннях ESC. 4. Проаналізувати лог польоту для точного налаштування фільтрів та PID-коефіцієнтів. Високий KV моторів для такого розміру пропелерів посилює проблему.
