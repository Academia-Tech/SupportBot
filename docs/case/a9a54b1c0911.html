Проблема

Un nuevo usuario encuentra TrueNAS demasiado complicado para su necesidad de copias de seguridad básicas de fotos, vídeos y documentos, y busca opciones más sencillas.

Рекомендація (не підтверджено)

Para un uso básico de NAS (copias de fotos, vídeos y documentos) sin la complejidad de TrueNAS, se recomienda considerar OpenMediaVault (OMV) o CasaOS. TrueNAS es potente pero exigente en recursos y puede ser complejo para usuarios principiantes que solo buscan acceso a carpetas. CasaOS se presenta como una opción muy sencilla y ligera, ideal para Docker. OMV es más robusto y flexible, adecuado para usuarios con más experiencia. Se menciona que muchos usuarios de Unraid están migrando a ZFS sin ECC, lo que añade contexto a las opciones de sistemas de archivos para NAS.
