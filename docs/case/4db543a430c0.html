Вибір Zigbee координатора (донгла) для Home Assistant

Case 4db543a430c0 · domotica_es

#zigbee#coordinator#dongle#home assistant#slzb-06#sonoff#ethernet#usb

Проблема

Користувач шукає USB Zigbee донгл для Home Assistant, оскільки вважає, що не може використовувати Ethernet-координатор (наприклад, SLZB-06), бо порт Ethernet на його міні-ПК зайнятий.

Рекомендація (не підтверджено)

Було надано роз'яснення та кілька рекомендацій: 1. Основна порада: Ethernet-координатори, як-от SMLight SLZB-06, не підключаються безпосередньо до HA-машини, а до будь-якого вільного порту в локальній мережі (наприклад, у роутері або світчі). Це дозволяє розмістити координатор у найкращому місці для покриття Zigbee. 2. SLZB-06 також має Wi-Fi як альтернативу Ethernet. 3. Якщо все ж потрібен USB-варіант, рекомендовано модель SMLight SLZB-07. 4. Також згадано Sonoff ZBBridge Ultra/Pro.
