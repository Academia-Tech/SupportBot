Connecting Tasmota Directly to InfluxDB on a Cloud Server

Case eb9d7f4ec8d3 · tasmota

#influxdb#tasmota#aws#cloud#database#integration#telegraf#mysql

Проблема

A user asked for help connecting a Tasmota device directly to an InfluxDB database hosted on an AWS EC2 instance. They were experiencing errors and another user suggested that connecting directly is improper and a gateway should be used instead, leading to a disagreement.

Рішення

The user who asked the question solved the problem on their own. They confirmed that it is indeed possible to connect Tasmota directly to InfluxDB without needing third-party applications like Telegraf or a separate gateway. They figured it out within a few hours and offered to explain the method to others via private message. In a follow-up clarification, user 'ouinouin' stated that Tasmota has a native plugin specifically for InfluxDB, but not for other databases like MySQL.
