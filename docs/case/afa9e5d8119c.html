Проблема

Користувач цікавиться, чи існують плагіни або інші інструменти для кастомізації та редагування елементів HUD (Heads-Up Display) у наземних станціях Mission Planner або QGroundControl.

Рішення

В якості альтернативної наземної станції з гнучким інтерфейсом рекомендовано спробувати Cockpit від Blue Robotics (https://github.com/bluerobotics/cockpit). Це ПЗ дозволяє налаштовувати відображення даних. Користувач підтвердив, що це 'офігенно' і його можна використовувати для OpenIPC, хоча воно не спеціалізоване для цього. Однак, варто враховувати, що Cockpit працює переважно з RTSP потоками, і може не підтримувати UDP стріми напряму. Як альтернативний підхід для керування відеопотоком, особливо з використанням Mavlink, було запропоновано mavlink-camera-manager (https://github.com/mavlink/mavlink-camera-manager), який може конвертувати формати стріму через gstreamer.
