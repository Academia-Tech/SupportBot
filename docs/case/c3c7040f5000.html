Проблема

Un usuario quiere crear una red Mesh inalámbrica en una casa de dos plantas usando un router Omada ER706W y dos APs adicionales, sin cableado, pero manteniendo funcionalidades avanzadas como VLANs, portal y VPN.

Рекомендація (не підтверджено)

La plataforma Omada está diseñada principalmente para configuraciones cableadas. Para redes Mesh inalámbricas, se sugieren otras marcas como Ubiquiti, Mikrotik o Ruijie Reyee, que ofrecen más opciones. También se recomienda considerar combos de nodos Mesh preconfigurados. El usuario busca una solución menos intrusiva que permita futuras mejoras con cableado y que soporte las funciones avanzadas de Omada.
