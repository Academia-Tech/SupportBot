Recibir notificaciones de llamada de Ring Intercom en HA

Case 32a7a661bf65 · domotica_es

#Ring Intercom#Home Assistant#notificaciones#automatización#timbre#eventos

Проблема

Un usuario busca cómo recibir notificaciones en Home Assistant cuando alguien llama al Ring Intercom, ya que el sensor de "última actividad" no es fiable.

Рішення

Para recibir notificaciones de llamada del Ring Intercom en HA, se recomienda usar un disparador (trigger) de tipo `state` para la entidad `event.verja_timbre`. Aunque la integración no permite hablar a través de HA, este evento se puede usar para disparar automatizaciones avanzadas, como capturar una imagen de una cámara cercana y enviarla por Telegram. Varios usuarios discuten la utilidad de esta integración, mencionando que, aunque la app de Ring es necesaria para hablar, la integración en HA permite crear un botón de apertura rápida en la pantalla de bloqueo (widget) y automatizar acciones que la app nativa no permite. Se menciona que la app de Ring sí tiene un widget de apertura, contrariamente a lo que un usuario creía.
