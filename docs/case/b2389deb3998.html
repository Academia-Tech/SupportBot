Can Tasmota emulate a matrix keyboard for an access control system?

Case b2389deb3998 · tasmota

#keyboard emulation#matrix keyboard#access control#custom script#Berry#ESP32#resistor ladder

Проблема

A user wants to use an ESP device with Tasmota to emulate a numeric matrix keyboard to send digit sequences to an existing access control board, without using multiple relays for each button.

Рішення

User 'ouinouin' explained that this is a complex task beyond standard Tasmota functionality. They clarified that such keyboards are typically a matrix or use a resistor ladder with an analog input, not individual switches. To mimic this, custom coding would be required. They suggested that the user would need to compile Tasmota with scripting support or use an ESP32 board with the Berry language to implement the custom logic for emulating the keyboard output. The original user accepted this explanation and thanked them.
