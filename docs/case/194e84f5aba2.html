Проблема

El usuario busca integrar un enchufe Tuya WiFi y un termómetro Tuya Bluetooth con Home Assistant, utilizando un coordinador SLZB-06, sin necesidad de un hub Tuya. El enchufe WiFi no aparece directamente en la red y el usuario duda sobre activar BLE en el SLZB.

Рекомендація (не підтверджено)

Para el enchufe Tuya WiFi, se recomienda usar la integración `LocalTuya` en Home Assistant, ya que la integración oficial de Tuya requiere la app y la nube. No es posible enlazar dispositivos WiFi Tuya directamente a Home Assistant sin una integración. Para el termómetro Tuya Bluetooth, se puede activar el BLE en el coordinador SLZB-06, aunque esto requiere un firmware específico para multiprotocolo (Zigbee y BLE) y puede eliminar la interfaz gráfica del SLZB-06, por lo que no se recomienda para evitar inestabilidad. Una alternativa más estable es usar un ESP32 con un reverse proxy dedicado para BLE (disponibles por unos 5€). Algunos dispositivos Bluetooth pueden tener el tráfico encriptado, requiriendo una clave para su integración.
