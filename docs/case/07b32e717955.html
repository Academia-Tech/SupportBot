Decisión sobre PC antiguo como NAS vs Ugreen 4800 Plus

Case 07b32e717955 · naseros

#NAS#PC antiguo#Ugreen#hardware#reutilización#OpenMediaVault

Проблема

Un usuario tiene un PC antiguo (i3-8100, 16GB RAM) que usó como HTPC/NAS y pregunta si debería venderlo para comprar un Ugreen 4800 Plus, usarlo tal cual, o actualizar componentes.

Рішення

Se recomienda reutilizar el PC antiguo, ya que sus especificaciones son más que suficientes para un NAS, con un consumo bajo. Se sugiere quitar la tarjeta gráfica si no es necesaria para reducir el consumo y usar la gráfica integrada. Otro usuario confirma que un i3 de 2ª generación con 4GB de RAM es suficiente para OpenMediaVault. El usuario decide darle una oportunidad al equipo existente.
