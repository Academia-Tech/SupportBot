Case 9cc1127a6700 · naseros

#Asus#router#Download Manager#HDD#formato de disco#NTFS#Ext4#solución de problemas

Проблема

Un usuario no puede instalar la aplicación Download Manager en su router Asus TUF AX6000. La instalación falla al 4% con un error de "no puede conectar con el servidor". El problema persiste incluso después de formatear el disco duro (HDD) conectado.

Рекомендація (не підтверджено)

Durante la conversación, se investiga el formato del disco duro. El usuario confirma que está en NTFS y que otros formatos como exFAT o FAT no son reconocidos por el router. Se sugiere que el problema podría ser el sistema de archivos. La recomendación final es que el usuario consulte la ficha técnica del router o busque en internet para verificar qué sistemas de archivos son compatibles con el router Asus para la función de Download Manager, sugiriendo probar con formatos de Linux como Ext2 o Ext4.
