Calibrating a Shelly EM1 with Two Clamps in Tasmota

Case eb5226b18d9b · tasmota

#Shelly EM1#power monitoring#calibration#Tasmota#clamp#PowerSet#CurrentSet

Проблема

A user flashed a Shelly EM1 with Tasmota and performed the power monitoring calibration procedure. After calibrating with a single clamp and getting perfect values, reconnecting the real system with both clamps resulted in unreal data for the second clamp (e.g., 1200 Amperes).

Рекомендація (не підтверджено)

For calibrating a Shelly EM1 with two clamps, it was suggested that the calibration commands need to be specified for each clamp individually. Instead of using generic commands like `PowerSet`, one should use commands with a suffix for the specific clamp, for example `PowerSet1` and `CurrentSet1` for the first clamp, and `PowerSet2` and `CurrentSet2` for the second. A further suggestion was to use the `PowerSetCal` command and to try copying the calibration data from the working clamp (clamp 1) to the problematic one (clamp 2) as a good starting point.
