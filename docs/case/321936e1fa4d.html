Instalar Home Assistant en Mini PC con Windows 11

Case 321936e1fa4d · domotica_es

#Home Assistant#Mini PC#Windows 11#VirtualBox#Proxmox#instalación#copias de seguridad

Проблема

Un usuario ha adquirido un Mini PC con Windows 11 y desea instalar Home Assistant, pero quiere seguir utilizando Windows para otras tareas. Pregunta si VirtualBox es la opción adecuada para esta configuración.

Рішення

Se confirma que VirtualBox es una buena opción para instalar Home Assistant en un Mini PC con Windows 11, permitiendo el uso concurrente de ambos sistemas. Se recomienda realizar copias de seguridad habituales, por ejemplo, en Google Cloud. Como alternativa superior, se recomienda encarecidamente instalar Proxmox en el Mini PC. Para seguir usando Windows, se puede instalar Windows 10 u 11 como una máquina virtual (VM) dentro de Proxmox, junto con otra VM para Home Assistant. Esta configuración es más estable para un servidor 24/7, ya que Proxmox está basado en Debian (Linux) y evita reinicios inesperados por actualizaciones de Windows. El usuario `user_1d812dc9b7` confirmó que investigaría esta opción.
