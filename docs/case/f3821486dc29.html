Вибір продуктивного роутера MikroTik та оцінка його IPsec продуктивності

Case f3821486dc29 · mikrotik_ua

#router#mikrotik#performance#ipsec#rb5009#ccr2216#ccr2116#mtu

Проблема

Користувач шукає роутер MikroTik з бюджетом до $1000, акцентуючи на високій продуктивності на одне ядро. Також виникає питання щодо реальної пропускної здатності моделі RB5009 при використанні шифрованих тунелів, зокрема IPsec.

Рекомендація (не підтверджено)

Надано список рекомендованих моделей: для максимальної продуктивності - CCR2216/CCR2116; як оптимальний вибір - RB5009 або CCR2004. Для найкращої одноядерної продуктивності запропоновано розглянути збірку на x86. Щодо продуктивності IPsec на RB5009, було підтверджено можливість досягнення гігабітної швидкості в тунелі IPIP+IPsec. Для цього достатньо вказати `IPsec Secret` в налаштуваннях IPIP-інтерфейсу та обов'язково зменшити MTU для уникнення втрати пакетів.
