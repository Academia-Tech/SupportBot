Проблема

Un usuario principiante pregunta cómo configurar su primer servidor usando un mini PC y un DAS. Sus objetivos incluyen instalar Home Assistant, una nube personal y un servidor Plex. Posteriormente, considera usar ZimaOS con un DAS TerraMaster.

Рекомендація (не підтверджено)

Se ofrecen varios consejos clave para principiantes que montan su primer servidor: 1. **Software**: Una buena opción es instalar un sistema operativo Linux y gestionar todos los servicios (Plex, Home Assistant) mediante contenedores Docker. ZimaOS también se menciona como una alternativa. 2. **Conexión DAS**: Es crucial no utilizar sistemas de archivos como RAID, BTRFS o ZFS sobre discos duros conectados por USB. 3. **Estrategia de backup**: Se enfatiza la importancia de tener una estrategia de copias de seguridad sólida, recordando que "un RAID no es un backup". Se debe planificar para el peor de los casos. Un ejemplo de buena estrategia es tener un NAS principal, un segundo NAS en una ubicación remota (ej. casa de un familiar) para copias nocturnas, y copias adicionales en discos USB. 4. **NAS vs. Nube**: Antes de invertir, se debe evaluar si un servicio en la nube como Google One podría ser más adecuado y económico para necesidades simples (ej. solo fotos y vídeos), ofreciendo simplicidad y tranquilidad sin la complejidad de gestionar un NAS.
