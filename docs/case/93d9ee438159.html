Case 93d9ee438159 · domotica_es

Проблема

Un usuario está desarrollando un sistema de alarma para Home Assistant con un lector de tarjetas NFC y un teclado conectado a un ESP, y busca información sobre integraciones de alarma en HA para usar sus sensores como disparadores. Expresa preocupación por la fiabilidad de las integraciones de HACS.

Рішення

Se recomienda encarecidamente utilizar la integración "Alarmo" disponible en HACS para crear un sistema de alarma en Home Assistant. Varios usuarios confirman que es muy completa, versátil y fiable. Como alternativa de mercado, se menciona el sistema de alarmas Ajax, que tiene integración nativa con Home Assistant, pero se advierte que es una solución mucho más costosa. Finalmente, el usuario que preguntó, a pesar de sus dudas iniciales sobre HACS, decide probar Alarmo.
