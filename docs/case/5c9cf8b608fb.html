Проблема

Un servidor con Proxmox 9 pierde la conexión de red por completo (ni localmente por ping), requiriendo un reinicio forzado. El problema apareció tras actualizar de Proxmox 8 y ejecutar un script de optimización de red "proxmenux".

Рекомендація (не підтверджено)

Se sospecha que el script de optimización ha causado el problema. Las posibles causas a investigar son: drivers de red corruptos, una opción de ahorro de energía en la BIOS/UEFI que esté desactivando la tarjeta de red, o un problema con el switch/router. Se recomienda revisar la configuración de administración de energía en la BIOS como primer paso.
