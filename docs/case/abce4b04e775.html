Проблема

Un usuario quiere instalar un sensor de temperatura y humedad en su garaje, pero su coordinador Conbee 2 en el salón no tiene suficiente alcance. Pregunta cómo extender la señal Zigbee, considerando que tiene un cable RJ45 que llega al rack del garaje.

Рекомендація (не підтверджено)

Para extender la señal Zigbee, se pueden usar enchufes o dispositivos Zigbee cableados que actúen como routers, aunque se advierte que si uno falla, la señal podría perderse. Otra opción es montar dos redes Zigbee separadas o usar alternativas como Bluetooth con ESPHome y Bluetooth Proxy. Se aclara que Zigbee y WiFi son redes distintas y no se extienden mutuamente. Se recomienda específicamente el SMLIGHT SLZB-06 en modo router. Aunque se conecta por Ethernet, su función de router Zigbee es inalámbrica. El puerto Ethernet/WiFi se usa para la configuración inicial y las actualizaciones, pero no es necesario para la conexión a Z2M/ZHA una vez configurado. Un usuario confirma que lo usa con USB y cable Ethernet, pero se aclara que el cable Ethernet no es imprescindible para el funcionamiento como router Zigbee, solo para gestión. Este dispositivo ofrece mayor cobertura y soporta hasta 160 dispositivos.
