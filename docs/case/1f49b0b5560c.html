Acceso a domótica con CGNAT y ONT sin admin

Case 1f49b0b5560c · naseros

Проблема

Un usuario desea acceder a sus dispositivos de domótica desde el exterior, pero su operadora lo tiene bajo CGNAT y no le proporciona acceso de administrador a la ONT.

Рекомендація (не підтверджено)

Para acceder a la domótica desde el exterior con CGNAT y sin acceso admin a la ONT, se sugieren varias soluciones: 1. **Salir del CGNAT**: Contactar con la operadora para solicitar una IP pública. Algunas operadoras lo ofrecen sin coste o por un pequeño cargo mensual (ej. Digi cobra 1€/mes). 2. **Utilizar software de VPN**: Opciones como ZeroTier o Tailscale permiten conectarse desde fuera sin necesidad de un VPS. 3. **Usar IPv6**: Si la operadora proporciona IPv6 nativo, los dispositivos tendrán una IPv6 pública, facilitando la conexión externa. 4. **Configurar túneles Wireguard**: Aunque no es una solución inmediata, permite conectar la red entera (o una VLAN) a través de un router más potente que los de las operadoras.
