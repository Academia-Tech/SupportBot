Рекомендація (не підтверджено)

Se buscan mirillas digitales discretas y con buena autonomía. Se recomienda una mirilla Tuya, aunque su batería dura poco con emisión continua. Para integración con Home Assistant y grabación por movimiento, se menciona que es difícil lograrlo localmente, aunque existen forks de go2rtc. Se añade la experiencia con el modelo EZVIZ CP4, que tiene buena autonomía de batería pero es estéticamente llamativo y no se integra con HA. Como solución para la estética, un usuario sugiere diseñar e imprimir una carcasa en 3D a medida para disimular tanto la cámara como el cableado.
