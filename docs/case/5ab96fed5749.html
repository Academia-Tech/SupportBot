UGREEN NAS no aparece en la red

Case 5ab96fed5749 · naseros

#UGREEN#NAS#Configuración#Red#DXP PLUS#Troubleshooting

Проблема

El usuario no logra que su unidad de red UGREEN 4800 DXP PLUS sea visible en la red después de cambiar de Terramaster a UGREEN.

Рішення

Para configurar el UGREEN NAS, es necesario instalar el software 'UGREEN NAS' (también conocido como 'UGREEN Network Assistant' o 'UGREEN DXP Finder'). Alternativamente, se puede intentar acceder a la página de configuración del NAS escribiendo 'find.ugreennas.com' en el navegador. El usuario confirmó que el problema fue resuelto.
