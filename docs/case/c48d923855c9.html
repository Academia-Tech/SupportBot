Cómo aislar cámaras IP para que solo funcionen en la red local con un router TP-Link

Case c48d923855c9 · naseros

#VLAN#firewall#ACL#TP-Link#Omada#ER7206#cámaras IP#seguridad

Проблема

Un usuario quiere aislar sus cámaras IP para que no tengan acceso a Internet, solo a la red local. Utiliza un router TP-Link ER7206 y pregunta cómo puede configurar esta restricción, sospechando que se necesita una regla de firewall.

Рішення

Se proponen dos soluciones principales para el router TP-Link ER7206: 1. Crear una VLAN dedicada para las cámaras, definir un perfil de switch para los puertos correspondientes y luego usar Listas de Control de Acceso (ACLs) para bloquear el acceso de esa VLAN a Internet. Se proporcionan enlaces a la guía de usuario de Omada y a ejemplos de configuración. 2. Como alternativa, en la VLAN de las cámaras, configurar el servidor DHCP para que asigne una puerta de enlace (gateway) a una IP que no exista en la red. Esto impedirá que los dispositivos de esa VLAN salgan a Internet. El usuario agradece ambas soluciones.
