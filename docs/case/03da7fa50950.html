Проблема

Un usuario pregunta sobre la vulnerabilidad a hackeos de una ONT Ubiquiti Nano G. La conversación deriva en una discusión sobre los posibles vectores de ataque a este tipo de dispositivos de red.

Рекомендація (не підтверджено)

Se explica que, si bien todo dispositivo es potencialmente vulnerable, un ataque a una ONT a través de vulnerabilidades de servidor web (como scripts PHP anticuados) es muy improbable, ya que las ONT no operan a ese nivel. Se menciona que un vector de ataque más plausible para equipos de operadora es el protocolo de gestión remota TR-069.
