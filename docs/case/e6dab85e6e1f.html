Проблема

El usuario busca opciones para domotizar un sistema de aire acondicionado centralizado con mando de pared, sin saber qué dispositivos usar.

Рішення

Para domotizar un aire acondicionado centralizado, se ofrecen varias soluciones: 1. **Control On/Off con Shelly 1**: Un usuario con un Fujitsu antiguo logró controlarlo solo para encendido/apagado conectando un Shelly 1 a los pines de la máquina. Esta solución es funcional y ha estado operando por años, aunque no permite control de temperatura o modos. 2. **Intesis Box (HMS Networks)**: Permite controlar todas las funciones del mando de pared (encendido, temperatura, etc.) conectándose al bus del mando. Sin embargo, un usuario reportó problemas de soporte con modelos antiguos. 3. **Solución ESP32**: Existe una opción basada en ESP32, con un video de YouTube como referencia para la implementación. 4. **Identificar marca y modelo**: Es crucial averiguar la marca y modelo del sistema de AC (abriendo el techo si es necesario) para encontrar la solución más adecuada.
