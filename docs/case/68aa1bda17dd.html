Проблема

Un usuario necesita controlar motores de persianas de 220V mediante un relé Zigbee, pero quiere usar pulsadores inalámbricos para evitar hacer rozas en la pared. Busca recomendaciones de pulsadores Zigbee que puedan pegarse a la pared.

Рекомендація (не підтверджено)

Se sugieren varias opciones de pulsadores inalámbricos Zigbee: 1. **Ikea RODRET:** Un regulador/interruptor inalámbrico con dos botones (arriba/abajo) que es visualmente claro para el control de persianas. 2. **Botón Zigbee de Xiaomi:** Un botón simple que puede configurarse para que cada pulsación cicle entre subir, parar y bajar. 3. **Botones de Ikea (otros modelos):** Se mencionan como una opción popular. 4. **Dispositivos de AliExpress:** Se indica que se pueden encontrar fácilmente pulsadores dobles en esta plataforma.
