Проблема

El usuario busca un disyuntor WiFi compatible con Home Assistant para sustituir un diferencial C25 y poder apagar remotamente el horno y la placa de inducción en un piso de alquiler turístico, por si los inquilinos los dejan encendidos.

Рекомендація (не підтверджено)

Para desconectar remotamente un horno o placa, no se debe sustituir el diferencial. Se recomienda usar un contactor normal (ej. Schneider) y controlarlo con un relé domótico (ej. Shelly 1). Una alternativa son los telerruptores para carril DIN (marcas como Chint o Tuya), que cambian de estado (ON/OFF) a cada pulsación. Para su uso, se debe configurar el switch para que realice la pulsación.
