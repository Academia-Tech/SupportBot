Case 174327dac029 · tasmota

Проблема

A user is unable to flash Tasmota on a Sonoff 4Ch Pro R3 using Tasmotizer on Linux; the process hangs after 'Uploading stub... Running stub... Stub running...'. Esptool.py also fails with a serial exception. The user suspects the serial adapter or power supply.

Рекомендація (не підтверджено)

To troubleshoot flashing issues with Sonoff 4Ch Pro R3: 1. Try using `esptool.py` directly with a slower baud rate, such as 115200. 2. Check the serial adapter and cable, as insufficient power or a bad cable can cause issues with baud rate stability. 3. Consider powering the Sonoff via its DC input while flashing, disconnecting the 3.3V from the adapter. 4. A common cause for flashing failures is low current from the USB port. 5. Refer to the Tasmota Esptool documentation for detailed instructions on putting the device in firmware upload mode: https://tasmota.github.io/docs/Esptool/#put-device-in-firmware-upload-mode.
