Case 4fa0fb8a1163 · naseros

#Masmovil#Fritzbox#ONT#PPPoE#IDONT#VLAN#Configuración

Проблема

Un usuario desea sustituir la ONT de MásMóvil por un router Fritzbox 5690 Pro, pero la operadora no le facilita el usuario PPPoE y la contraseña.

Рішення

Para obtener los datos necesarios (IDONT/GPON SN) de MásMóvil y usar un router propio como un Fritzbox, se confirmó que la siguiente solución es efectiva. Se debe simular una avería para que acuda un técnico y le facilite los datos. La configuración para MásMóvil no usa PPPoE, sino DHCP, y requiere el número de ONT (IDONT o PLOAM) y configurar la VLAN 20 con pbit 0. El usuario que preguntó confirmó que la información fue de gran ayuda.
