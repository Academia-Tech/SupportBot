How to Upgrade Tasmota Firmware on Gosund EP2 Sockets

Case ddba93a45877 · tasmota

#upgrade#firmware#OTA#Gosund EP2#Tasmota#update path#web UI

Проблема

A user with Gosund EP2 sockets running Tasmota v9.2.0 was unsure about the safety and procedure for upgrading to the latest version. They were looking for guidance on the correct upgrade path.

Рішення

It is safe to upgrade from Tasmota v9.2.0. For versions 9.1 and above, a direct upgrade to the latest release is supported by using the "Online Update" button in the web UI. Tasmota handles the required two-step process (flashing a minimal build, then the full build) automatically. During the upgrade, it is important to be patient and not refresh the browser. The process was confirmed to work successfully by another user.
