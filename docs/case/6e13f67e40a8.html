Проблема

Un usuario busca consejo para elegir entre dos opciones para montar un servidor con Proxmox: un Mini PC con procesador Intel N100 o un HP EliteDesk 800 G2. El objetivo es tener un equipo potente para aprender y experimentar, ya que dispone de otro servidor dedicado exclusivamente para Plex.

Рекомендація (не підтверджено)

Se ofrece una comparativa detallada entre el Mini PC Intel N100 y el HP EliteDesk 800 G2 para un servidor Proxmox. El HP EliteDesk 800 G2 destaca por su mayor capacidad de expansión, permitiendo instalar más memoria RAM y, posiblemente, cambiar la CPU, lo que lo hace más adecuado si se planea ejecutar muchas máquinas virtuales o servicios que requieran más recursos. Por otro lado, el Mini PC con Intel N100 es mucho más eficiente energéticamente y tiene una mejor GPU para funciones de centro multimedia. Si el uso principal es para trastear, montar Pi-hole, una nube personal y Home Assistant, el N100 es suficiente, especialmente con 16GB de RAM. Sin embargo, se aconseja planificar bien el uso futuro, ya que la ampliación posterior de un N100 podría ser más costosa que invertir directamente en un HP si se prevé una carga de trabajo mayor. Se aclara que para transcodificación de Plex, se necesita una GPU dedicada, no CPU ni RAM, y que servicios como Pi-hole o Home Assistant no impactarán significativamente el rendimiento de transcodificación.
