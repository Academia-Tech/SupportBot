#rf#rolling code#código evolutivo#persianas#estores#tuya#gaviota#home assistant

Проблема

Un usuario no consigue replicar un mando a distancia RF con Rolling Code para un estor motorizado. Los dispositivos universales que ha probado no funcionan, ya que no pueden almacenar y replicar el código variable.

Рекомендація (не підтверджено)

Se proponen dos soluciones basadas en la experiencia de otros usuarios: 1. Utilizar un mando universal IR+RF de Tuya Smart. Aunque la copia directa puede fallar, este dispositivo logró funcionar para un toldo Gaviota. La integración en Home Assistant se realiza a través de escenas de Tuya. 2. Para persianas y toldos de la marca Gaviota, comprar el módulo de control oficial del fabricante, que se integra directamente y sin problemas con Home Assistant.
