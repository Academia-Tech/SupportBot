Sonoff 4Ch R3 PRO Detected as Sonoff Basic After Flashing

Case f324389c4566 · tasmota

#Sonoff 4CH Pro R3#Tasmota#template#configuration#Sonoff Basic#multi-channel#web UI

Проблема

A user flashed a Sonoff 4Ch R3 PRO with Tasmota using Tasmotizer, but the device was identified as a Sonoff Basic in the web UI. This resulted in only one of the four channels being controllable.

Рішення

The issue was resolved by manually selecting the correct device template in the Tasmota web UI. The correct procedure is to navigate to "Configuration" -> "Configure Module" and select the "Sonoff 4CH Pro R3 (7)" module from the list. A link to the correct template on the Blakadder website (https://templates.blakadder.com/sonoff_4CHPROR3.html) was also provided as a reference.
