Неможливість отримати телеметрію з ESC Hobbywing 150A v5 в ArduPilot

Case 42d5a00ab357 · ua_ardupilot

#hobbywing#esc#телеметрія#ardupilot#протокол#t-motor#data link#s.port

Проблема

Користувач намагається отримати дані телеметрії (напруга, струм) з регулятора Hobbywing 150A v5 на польотний контролер з ArduPilot. Стандартні налаштування протоколу (SERIALx_PROTOCOL = 16) не працюють.

Рекомендація (не підтверджено)

ArduPilot наразі не підтримує пропрієтарний протокол телеметрії від ESC Hobbywing та T-motor напряму. Для отримання даних потрібен додатковий апаратний перетворювач, такий як "data link" або "s.port". Було надано посилання на відкритий тікет на GitHub, що підтверджує цю проблему: https://github.com/ArduPilot/ardupilot/pull/22649. Як альтернатива, прошивка Rotorflight підтримує ці ESC без додаткових пристроїв.
