No se puede iniciar sesión en Home Assistant con Chrome después de una actualización

Case 062722ca803e · domotica_es

#Home Assistant#Chrome#Login#2FA#Extensión#Navegador#Solución de problemas

Проблема

Un usuario no puede acceder a su Home Assistant a través del navegador Chrome en su PC principal después de actualizar el core. El proceso de inicio de sesión se queda cargando indefinidamente después de introducir la contraseña y el 2FA, aunque sí funciona en modo incógnito y en otros navegadores como Edge. Restaurar copias de seguridad anteriores no soluciona el problema.

Рішення

El problema fue resuelto por el propio usuario, quien descubrió que el conflicto era causado por una extensión del navegador Chrome. Al deshabilitar o gestionar las extensiones, pudo volver a iniciar sesión correctamente.
