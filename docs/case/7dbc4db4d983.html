Tasmota Devices Lose MQTT Credentials After Power Loss

Case 7dbc4db4d983 · tasmota

#power loss#mqtt#credentials#configuration loss#savedata#troubleshooting

Проблема

A user reported that after a brief power outage, several of their Tasmota devices (versions 9.4.0 to 12.5.0) lost their configured MQTT username and password, while other settings remained intact. The devices were still connected to the network.

Рекомендація (не підтверджено)

Other users confirmed this is not normal behavior. One user suggested investigating the `savedata` status on an affected device, as this setting controls how often configuration changes are written to flash memory and could be related to the issue. The original user had already resolved the immediate problem by manually re-entering the credentials on all devices.
