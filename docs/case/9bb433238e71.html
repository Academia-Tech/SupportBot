Проблема

Обговорюються різні підходи до точного калібрування сенсора струму. Існуючі аналогові датчики можуть мати неточності (offset), а дані з ESC не враховують споживання периферії. Виникає питання, як отримати надійні показники споживання енергії для оцінки залишку заряду батареї.

Рекомендація (не підтверджено)

Для калібрування сенсора струму рекомендовано декілька методів: 1. **Зовнішній ватметр (Power Meter):** Підключити між батареєю та дроном для вимірювання реального споживання в польоті або під навантаженням. 2. **Струмові кліщі (для постійного струму):** Найпростіший спосіб для вимірювання струму на вході 4-в-1 ESC. 3. **Регульоване навантаження:** Дозволяє відкалібрувати датчик на конкретному, наприклад, круїзному струмі для максимальної точності в цьому режимі. 4. **Телеметрія з ESC (DShot):** Використання цифрових даних з регуляторів є точним, але не враховує споживання периферії (польотний контролер, VTX тощо). Також важливо не забувати про попереднє калібрування датчика напруги.
