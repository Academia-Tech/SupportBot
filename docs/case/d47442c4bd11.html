Viabilidad de instalar Home Assistant en una Raspberry Pi 3 B+

Case d47442c4bd11 · domotica_es

#Home Assistant#Raspberry Pi#RPi 3#instalación#hardware#rendimiento

Проблема

Un usuario pregunta si es viable instalar Home Assistant en una Raspberry Pi 3 B+ y si el hardware es suficiente para correr el sistema de forma estable.

Рішення

Se confirma que es posible instalar Home Assistant en una Raspberry Pi 3 B+, pero con limitaciones significativas. Se recomienda para empezar con sensores y switches, pero se advierte que el rendimiento será bajo y no es adecuado para un uso intensivo. Es importante asegurarse de que la placa sea de 64 bits para poder instalar la versión soportada de HAOS. Finalmente, el usuario, tras recibir el consejo, optó por instalarlo en un mini PC para obtener un mejor rendimiento.
