Проблема

Користувач шукає рішення для подовження Type-C підключення DJI Goggles 2 та передачі відеосигналу на значну відстань, відкидаючи варіант розміщення пульта на щоглі через незручність та швидкий розряд батареї.

Рішення

Було запропоновано кілька рішень для подовження підключення DJI Goggles 2 та передачі відео: 1. Використання Cosmostreamer (cosmostreamer.com/products/gd1/), що було прийнято користувачем як гарна ідея. Користувач user_39d0b3fa6b описав детальний реалізований сетап: антена Phoenix, окуляри з виведеними SMA-виходами підключаються до антени, антена живиться від окремої батки. Окуляри під'єднуються до Cosmostreamer (з додатково розпаяним BEC для живлення окулярів), оптичний HDMI кабель потрібної довжини йде до підвалу, де підключається до цифрової плати відеозахвату, від неї USB до ноутбука для запису/трансляцій та HDMI до монітора. Управління йде по витій парі з антени до пульта окремо. 2. Застосування USB-подовжувачів по витій парі (наприклад, Digitus USB 2.0 - UTP Cat5), з можливістю пошуку USB-C версій або використання перехідників. 3. Розглядався варіант використання одноплатника з Android та відповідною версією DJI Fly замість Cosmostreamer. Користувач user_c5b07ab326 зазначив, що адекватно працює тільки Raspberry Pi 5, але для навігації інтерфейсом потрібна мишка, що вимагає додаткової лінії на мачту, хоча існують подовжувачі HDMI + USB по одному кабелю. Обговорювалося питання живлення пульта через Type-C: хоча один порт використовується для зарядки та виводу, можливо створити "хитрий" кабель або припаятися безпосередньо до батарей пульта, але це складно.
