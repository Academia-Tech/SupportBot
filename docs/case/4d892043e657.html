Проблема

Un usuario pregunta sobre la mejor opción de cableado de red (Cat 7 o Cat 8) para una obra nueva en una vivienda particular, considerando si merece la pena la inversión.

Рекомендація (не підтверджено)

Para el cableado de red en una vivienda, se debate entre cable de cobre (UTP) y fibra óptica. El cable Cat 6A es una opción robusta que soporta 10Gbps y permite el uso de PoE. La fibra óptica ofrece inmunidad a interferencias, es más delgada y económica, pero se considera poco práctica para cablear cada toma final debido a la necesidad de conversores o switches con puertos SFP, lo que incrementa el coste y la complejidad, además de no soportar PoE directamente. La recomendación general es un enfoque híbrido: utilizar fibra óptica para los enlaces troncales (ej. entre switches de diferentes plantas) y cable UTP Cat 6/6A para la conexión final a los dispositivos en las habitaciones.
