Пошук плагінів або способів редагування HUD в Mission Planner/QGC

Case c78673c231f1 · ua_ardupilot

#HUD#Mission Planner#QGC#Cockpit#GCS#кастомізація#інтерфейс

Проблема

Користувач шукав плагіни або інші способи для редагування та кастомізації інтерфейсу HUD (Heads-Up Display) у наземних станціях Mission Planner або QGC.

Рішення

Оригінальне рішення для пошуку плагінів або способів редагування HUD в Mission Planner/QGC полягало у використанні альтернативної наземної станції Cockpit від Blue Robotics (https://github.com/bluerobotics/cockpit), яка надає більше можливостей для кастомізації інтерфейсу. Додатково, користувач знайшов рішення для відображення прицільної сітки, встановивши програму "hudsight" для комп'ютерних ігор. Також виникла проблема з зависанням QGroundControl та надмірним споживанням пам'яті процесора під час відеострімінгу. Для вирішення цієї проблеми було рекомендовано встановити налаштування "Low latency" в QGC.
