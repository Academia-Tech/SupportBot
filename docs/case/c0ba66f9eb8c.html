Проблема

Un usuario consulta si es factible acortar la tija de 15 cm de un ventilador de techo de la marca Create para que quede más pegado al techo.

Рекомендація (не підтверджено)

Se indica que es técnicamente posible cortar la tija, siempre que se deje espacio suficiente para la instalación y se pueda perforar un nuevo agujero para el pasador de seguridad. Sin embargo, se advierte que reducir la distancia mínima al techo recomendada por el fabricante podría generar ruidos aerodinámicos no deseados debido a la falta de espacio para el flujo de aire.
