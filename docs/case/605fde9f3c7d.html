Creating a Momentary Low Pulse on a GPIO Pin

Case 605fde9f3c7d · tasmota

#gpio#pulsetime#momentary#esp-01s#relay_i#inverted#pulse

Проблема

A user needed to configure a GPIO pin on an ESP-01s to be normally HIGH (3.3V) and then go LOW (0V) for a second upon activation, before returning to HIGH.

Рішення

The solution involves two steps: 1. Use the `PulseTime` command to make the output automatically turn OFF after a set duration. A guide was provided: https://nerdiy.de/en/tasmota-nachlaufen-eines-relais-einstellen/ 2. To achieve the desired HIGH -> LOW -> HIGH logic (instead of the default LOW -> HIGH -> LOW), configure the GPIO as an inverted relay (`Relay_i`) in the device's template. The user confirmed both `PulseTime` and the inverted logic worked.
