Considerations for Updating Tasmota on Sonoff Zigbee Bridge Pro

Case df56649dea04 · tasmota

#Tasmota#update#Sonoff Zigbee Bridge Pro#firmware#ESP32#backup

Проблема

A user flashed their Sonoff Zigbee Bridge Pro with Tasmota two months ago and is now considering an update, asking what precautions to take.

Рішення

When updating a Sonoff Zigbee Bridge Pro, the recommended order is to update the Tasmota firmware first, and then the Zigbee coordinator module firmware. A user confirmed a successful upgrade to Tasmota v13 and coordinator firmware `SonoffZBPro_coord_20230507.hex`. During this process, it was noted that with Tasmota v13, the output log from the Berry console is now shown in the default Tasmota console. After the upgrade, the user experienced a connection error with Home Assistant. The problem was resolved by performing a new initial flash of Tasmota using a serial TTL adapter. The user also noted that partition resizing was not necessary when flashing Tasmota v13.
