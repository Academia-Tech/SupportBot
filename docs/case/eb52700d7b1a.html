Problemas con la integración 'remote_homeassistant' y la importación masiva de entidades

Case eb52700d7b1a · domotica_es

#home assistant#integración#remote_homeassistant#entidades#configuración#HACS#YAML

Проблема

Un usuario tiene problemas para configurar la integración `remote_homeassistant`. Inicialmente no le aparecen entidades y, tras varias pruebas, importa accidentalmente cientos de entidades no deseadas desde la instancia remota, las cuales además no puede borrar.

Рішення

La solución consiste en configurar correctamente la integración para filtrar qué entidades se importan, en lugar de importarlas todas. Al especificar qué dominios o entidades se desean y reiniciar, las entidades no deseadas que se habían importado previamente se desbloquearon, permitiendo su eliminación manual. Se aclara que el parámetro `service_prefix` sirve para añadir un prefijo y diferenciar las entidades remotas (ej. `chalet_`).
