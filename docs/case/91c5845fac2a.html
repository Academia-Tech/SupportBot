Cómo instalar un Shelly para controlar una depuradora de piscina con contactor

Case 91c5845fac2a · domotica_es

#shelly#piscina#depuradora#contactor#cableado#domótica#motor

Проблема

Un usuario solicita ayuda para instalar un Shelly en el cuadro de una depuradora de piscina, con el objetivo de domotizarla sin perder el selector manual/automático existente. Se discute el cableado, la necesidad de un contactor y se proporcionan esquemas.

Рекомендація (не підтверджено)

Se recomienda encarecidamente el uso de un contactor para proteger tanto el Shelly como el motor de la bomba, ya que el consumo del motor puede dañar el relé. Varios usuarios confirman que esta es la mejor práctica. Un usuario proporciona instrucciones detalladas de cableado: alimentar el Shelly (L+N), llevar fase a la borna I (In) del Shelly, y conectar la borna O (Out) a la bobina A1 del contactor, con A2 a neutro. Adicionalmente, se explica cómo mantener el reloj programador antiguo como sistema de redundancia (por si falla el internet), conectando sus bornes en paralelo a las bornas I y O del Shelly. Se aclara que el Shelly puede funcionar de forma autónoma sin internet una vez programado. Finalmente, se comparte un esquema de conexión como referencia visual: https://images.app.goo.gl/s2fUzWwCpTH9D6DW9.
