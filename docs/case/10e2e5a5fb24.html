Проблема

Поділено скрипт для ArduPilot, що дозволяє вивести коптер із зони втрати відео або керування за заданим курсом, актуальний для бомберів, крил та НРК.

Рішення

Скрипт для ArduPilot, що забезпечує виведення коптера з зони втрати відео чи керування за заданим або визначеним курсом. Скрипт облітаний з повним вимкненням пульта, працює передбачувано: при втраті лінку дрон набирає висоту і летить зазначеним курсом до відновлення зв'язку. Вхід та вихід в фейлсейф працює. Альтернативою Lua-скрипту для вимкнення GPS в місії є використання дефолтної команди `DO_AUX_FUNCTION`. Для вимкнення GPS потрібно встановити номер команди 65 та стан -2, для увімкнення - 0. Ця функція дозволяє вимикати GPS на певній точці місії без використання Lua-скриптів. Інформацію про цю команду важко знайти в документації, її було виявлено при аналізі коду Mission Planner та ArduPilot. Користувач підтвердив, що спробує це рішення.
