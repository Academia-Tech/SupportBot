Проблема

Un usuario busca un NAS para gestionar su colección de fotos y reducir su dependencia de servicios en la nube como Google Photos. Duda entre un Synology DS923+ y un Ugreen DXP4800 Plus, y busca la mejor opción para este propósito.

Рекомендація (не підтверджено)

Se ofrecen varias recomendaciones según el perfil del usuario: - Para usuarios que no quieren complicaciones y buscan una solución fácil de usar, se recomienda Synology, ya que su aplicación Synology Photos es considerada la mejor para la gestión de fotos en NAS comerciales. - Para usuarios más técnicos o que buscan mayor flexibilidad, se recomienda Ugreen, especialmente si se planea usar aplicaciones como Immich. Se aclara que no es posible guardar el backup automático de WhatsApp directamente en un NAS. Respecto a la seguridad, se indica que el nivel es similar entre marcas si se siguen buenas prácticas (no exponer servicios a internet, contraseñas fuertes, doble factor). El usuario expresa preocupación por las restricciones de Synology en cuanto a discos y memoria RAM no oficiales, lo que le hace considerar el Ugreen DXP4800 Plus como alternativa al DS923+. La elección final depende del tipo de implementación deseada (facilidad de uso vs. personalización y servicios propios). Para recomendaciones de discos duros, se sugiere consultar el canal #todo-sobre-hdd.
