Cómo sustituir el router de Pro-Digi por uno propio (configuración estándar y 10Gbps)

Case aac5dea08702 · naseros

#pro-digi#digi#router#pppoe#vlan#ont#sfp+#10gbps#asus

Проблема

Un usuario quiere contratar Pro-Digi y pregunta cómo puede utilizar su propio router (Asus) en lugar del que proporciona el operador, tanto para conexiones estándar como para la de 10Gbps, evitando soluciones como DMZ.

Рекомендація (не підтверджено)

Se confirman varias soluciones: 1. **Para conexiones estándar (no 10G)**: Es posible usar un router propio (ej. Asus XT12). La configuración clave es cambiar el modo de conexión a PPPoE y explícitamente NO crear la VLAN 20. 2. **Para conexiones de 10Gbps (sustitución completa)**: Se necesita un router con al menos dos puertos de 10Gb (uno SFP+), con aceleración por hardware para PPPoE. Se recomienda una ONT SFP+ (ej. de fs.com por ~250€ o de AliExpress por ~150€). Un usuario confirma tener un Asus BE88U con ONT SFP+ funcionando perfectamente. Se advierte que si el router solo tiene puertos RJ45 de 10Gb, una ONT SFP+ no sería compatible directamente, y se necesitaría una ONT con salida RJ45 de 10Gb compatible con Digi, de las cuales no se conocen opciones.
