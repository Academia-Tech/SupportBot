Проблема

Користувач шукає всепогодну камеру для Raspberry Pi, яку можна використовувати для ретрансляції сигналу з мінімальною затримкою. Існуючі IP-камери мають значний лаг.

Рекомендація (не підтверджено)

Для Raspberry Pi рекомендовано модуль камери Waveshare IMX290-83 IR-CUT (посилання: https://evo.net.ua/modul-kamery-waveshare-imx290-83-ir-cut/?srsltid=AfmBOorq1aaY_Nzg5Q9v852BqgDB4zKhwu9Z26_xQ191VpPvC8EnOnc-). Вона не є всепогодною, тому потребує 3D-друкованого корпусу. Як альтернатива, запропоновано використовувати IP-камери, прошиті OpenIPC, які забезпечують практично відсутність затримки. Користувач підтвердив, що IP-камери (наприклад, Hikvision) є простішим рішенням, оскільки вони стрімять відео на іншу IP-адресу без необхідності прокидання через Raspberry Pi. Raspberry Pi буде використовуватися лише для Mavlink.
