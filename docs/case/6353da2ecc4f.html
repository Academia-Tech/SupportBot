Проблема

El usuario experimenta velocidades de descarga inestables (subidas y bajadas, pausas prolongadas) con fibra Digi, especialmente al descargar actualizaciones de Fortnite. Reiniciar el router (Zyxel EX3501-T0) soluciona el problema temporalmente. El problema no ocurre en todas las computadoras y no cree que el router esté saturado con solo 10 dispositivos.

Рекомендація (не підтверджено)

Se sugiere que la velocidad de descarga inestable con el router Zyxel EX3501-T0 de Digi podría deberse a una saturación o una falla física del equipo. Como solución, se recomienda contactar al proveedor (Digi) para solicitar un reemplazo del router. El usuario confirmó que se pondría en contacto con ellos para gestionar el cambio.
