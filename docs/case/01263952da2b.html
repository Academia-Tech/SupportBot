Проблема

Пошук рішення для передачі телеметрії (керування) та відеосигналу з борту БПЛА на наземну станцію через супутниковий інтернет Starlink.

Рекомендація (не підтверджено)

Запропоновано кілька підходів: 1. **DIY з одноплатним комп'ютером:** На борту встановлюється одноплатний комп'ютер (напр. Raspberry Pi), який пакує UART-телеметрію через `ser2net` і відеопотік через RTSP-сервер для передачі через Starlink. На землі процес зворотній. 2. **Готові програмні рішення:** Рекомендовано використовувати готові платформи, такі як BlueOS (blueos.cloud), які мають вбудовані інструменти для трансляції відео та керування через веб-інтерфейс (Cockpit). 3. **Спеціалізовані системи:** Згадано проект r2d2.org як приклад готового рішення для подібних завдань, особливо в умовах РЕБ.
