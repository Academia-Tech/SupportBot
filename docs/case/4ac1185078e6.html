Проблема

Un usuario pregunta sobre la experiencia con el Synology DS223j. Quiere usarlo para reemplazar Google Fotos e iCloud, y para montar un servidor de Plex. Le preocupa el rendimiento del dispositivo, especialmente con Plex y la transcodificación de vídeo.

Рішення

Se desaconseja el uso del Synology DS223j para servicios como Plex si se requiere transcodificación de vídeo, ya que su CPU y RAM son limitadas. Para un uso centrado en servicios, se sugiere un modelo superior como el DS224+. Sin embargo, el DS223j es una opción viable para iniciarse si se tienen claras sus limitaciones. Funciona bien para tareas como Synology Photos, copias de seguridad y almacenamiento, especialmente si los dispositivos de reproducción son modernos (ej. iPhone 11/15, iPads) y compatibles con códecs como H.265, evitando así la necesidad de transcodificar. Una estrategia de crecimiento rentable es comenzar con el DS223j y, si en el futuro se necesita más potencia para servicios, añadir un miniPC dedicado para ello, en lugar de invertir inicialmente en un NAS más caro. Para el almacenamiento, se puede empezar con un solo disco y añadir un segundo más adelante para ampliar o crear un RAID.
