Проблема з копіюванням радіочастотного (RF) пульта з Rolling Code

Case 97b084db5097 · domotica_es

#RF#Rolling Code#Tuya#remote control#blinds#awning#Gaviota#Home Assistant

Проблема

Користувач не може скопіювати або реплікувати сигнал від RF-пульта для моторизованих ролет, оскільки він використовує технологію Rolling Code. Стандартні пристрої для копіювання RF-сигналів не працюють, оскільки код постійно змінюється.

Рекомендація (не підтверджено)

Для вирішення проблеми з RF Rolling Code було запропоновано два підходи: 1. Використовувати універсальний пульт Tuya Smart IR+RF. Один з користувачів підтвердив, що йому вдалося вирішити аналогічну проблему з тентом Gaviota за допомогою такого пристрою. Інтеграція в Home Assistant здійснюється через сцени Tuya. 2. Придбати офіційний модуль від виробника ролет (наприклад, Gaviota), який зазвичай має пряму інтеграцію з Home Assistant.
