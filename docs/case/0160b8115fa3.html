#zigbee#button#long press#pairing mode#moes#aqara#philips hue#home assistant

Проблема

Користувач підтвердив, що нова кнопка Moes також має проблему з довгим натисканням, яке активує режим сполучення. Подальше обговорення виявило ключову різницю: кнопки Aqara мають окрему фізичну кнопку для сполучення, тоді як Moes та інші дешеві аналоги використовують для цього довге натискання основного перемикача.

Рекомендація (не підтверджено)

Підтверджено, що некоректна робота 'long press' є конструктивною особливістю кнопок без окремого режиму сполучення. В Home Assistant кнопки Aqara надають події 'on_press', 'on_release', що дозволяє гнучко налаштовувати автоматизації, тоді як у кнопок Moes/AliExpress таких можливостей немає. Остаточна рекомендація — обирати кнопки з окремим механізмом сполучення, такі як Aqara або Philips Hue, для надійної роботи всіх типів натискань.
